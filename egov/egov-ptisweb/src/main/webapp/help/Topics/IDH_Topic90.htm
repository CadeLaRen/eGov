Deactivating a Property

 

You can deactivate a property when:

·    there is a case pending against it

·    a wrong or duplicate property is created


NOTE: You cannot perform any action on a property that has been deactivated. However, you can view and search the deactivated property.

 

To deactivate a property, perform the following steps:

1.   Find the property which you want to deactivate.

2.   From the Actions drop-down list, select Deactivate Property to display the DeActivate Property page.

3.   From the Reason for Deactivation drop-down list, select one of the following options:

·    Wrong Entry in System: If you want to deactivate the property because it has been wrongly created.

·    Court Case: if you want to deactivate the property because there is a case pending against it.

4.   In the Order No. field, type the order number of the property. Format: D/Zone/Ward/Financial Year/Sequence Number.

Example: D/01/029/09-10/0019.

5.   In the Order Date field, type the order date.

6.   In the Comments field, type the comments regarding the deactivation of the property.

7.   Click Upload Document to upload a required document.

8.   Click Forward to Forward the application to high authorities.                

9.   Click Approve to Approve the changes.

10. Click Close to exit from the screen.
