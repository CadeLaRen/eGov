Mutation of Property

 

 In case of Mutation of an Property, perform the following steps:

1.   Find the property  which you want to change the address of.

2.   From the Actions drop-down list, select Mutation to display the Property Particulars page.

3.   Enter the details which you want to change.

 

                                        Table : Mutation Details

Field

Description

Reason for Mutation

From the reason for Mutation drop down list select appropriate reason

Date of application

Enter Date of the application

Sale Details

Enter details for selling the property

Mutation Date

Enter date of mutation

Court Order Number

Enter Court order number if you select court order in reason for mutation

Name of the sub register office

Enter the name of sub register office

Document Number

Enter Number of the Document

Document Date

Enter Document date

Mutation Fee

Enter Mutation fee

Receipt Number

Enter Receipt Number

MOwner Name

Enter owner name.

Note: click on plus symbol to add more owner names.

Mobile number

Enter Mobile number of the owner

EMail Address

Enter Email id of the owner.

 

 

 

3.   Click Forward to Forward the application to high authorities.                

4.   Click Approve to Approve the changes.

5.   Click Close to exit from the screen.
