Change Location Rate

 

To Change Location rate of a property, perform the following steps:

1.   Click Property Tax from the left navigation bar.

2.   Click Administration.

3.   Click Change Location rate to display the Change location rate page.

4.   Do the following steps:

 

                                        Table 1: Property Bounded By

Field

Description

Zone

From the Zone drop-down list, select the zone to which the property belongs

Ward

From the Ward drop-down list, select the ward to which the property belongs

Area

From the Area drop-down list, select the area to which the property belongs
