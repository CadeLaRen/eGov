Finding a Property

 

You can find a registered property using any one of the following:

·     Bill Number

·     Mobile Number

·     Zone and Ward

·     Area and Location 


To find a property, perform the following steps:

1.   Click Property Tax from the left navigation bar.

2.   Click Existing Property.

3.   Click Search Property to display the Search Property page.

4.   Do one of the following steps:

·    To find a property using the Index Number, do the following steps:

a. In the Index Number field, type the Index number to which the property belongs.

b. In the Parcel Id field, type the Parcel Id to which the property belongs to.

 

·    To find a property using the Zone and Ward, do the following steps:

a. From the Zone drop-down list, select the number of the zone to which the property belongs.

b. From the Ward drop-down list, select the ward number to which the property belongs.

c. In the Plot No/House No. type the Plot or House number of the property.

d. In the Owner Name field, type the name of the property owner.

 

·    To find a property using the Owner Name, do the following steps:

a. In the Area drop-down list, select name of teh area to which the property belongs.

b. In the Owner Name field, type the name of the owner of the property.

c. In the Plot No/House No. type the Plot or House number of the property.

d. In the Old Number field, type old new door number of the property.



·    To find a property by using the Objection details, do the following steps:

a. From the Property Type drop-down list, select the Property of which type it belongs.

b. In the Objection Number field, type the objection number of the property.

e. In the From Date field, enter the from date of the property.

f.  In the To Date field, enter the To date of the property.

g. Click Search to display the Search Results page.
