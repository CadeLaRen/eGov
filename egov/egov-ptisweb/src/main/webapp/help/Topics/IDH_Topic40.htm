Creating a New Property

 

You can register a new property when:

·      the assessor discovers a new unassessed property during field survey and gets the correct property details.

·      citizens inform the corporation about a property which is yet to be registered in their names.

·      if the property details are not available in the PTIS system a citizen can also create a new property. To check the availability of the property details in the PTIS system, see Finding a Property.

 

To register a new property, perform the following steps:

1.  Click Property Tax from the left navigation bar.

2.  Click New Property.

3.  Click Create New Property Workflow to display the Create Property: Form-6  page.

4.  From the Reason for Creation of Property drop-down list, select NEW PROPERTY.

5.  Specify the following property details:

 

                                        

                                        Table 1: Create Property

Field

Description

Zone

The zone to which the property belongs.

Ward

The ward to which the property belongs.

Owner Name

The name of the property owner(s). 


NOTE: 

·       If the property has multiple owners, click the Plus (+) icon and type the name of the additional owner(s). Similarly, click the Cross (X) icon to delete the selected owner.

 

·      You must type the name of the owner exactly as it is given in the form submitted. Once you register the property with a wrong name, you cannot change it. 

 

Mobile Number

The mobile phone number of the property owner.

E-mail Address

The e-mail address of the property owner.

 

6. In the Property Address section, specify the following property details:

                                        Table 2: Property Address

Field

Description

Area

The area to which the property belongs.

Plot No/House No

The House number of the property

Old Number

The House number that was earlier assigned to the property.

Address

The address of the building or apartment.

Pincode

the postal code with first three digits as 440. For example: 440123

Khasra Number

Enter the khasra no.

Mauza

Enter Mauza

City Survey Number

Enter the city survey number

Sheet Number

Enter sheet number

Parcel Id

Enter Parcel Id

Reason for Tax Exemption

Enter Reason for Tax Exemption

7. In the Property Bounded By section, specify the following property details:

                                        Table 3: Property Bounded By

Field

Description

North Ward

The property number bounded to the north of the property being registered.

South Ward

The property number bounded to the south of the property being registered.

East Ward

The property number bounded to the east of the property being registered.

West Ward

The property number bounded to the west of the property being registered.

 

8. Select the type of the property from the Property Type drop-down list, and perform the following steps:

·         Fields which are common in all Property type details:

                    Table 3: Property Type

Field

Description

Property Category

From the Property category drop down list select appropriate Property

Area of Plot

Enter Area of the plot in Sq.mtrs

Location Factor

From the Location factor drop down list select appropriate location. 

General water rate

From the general water rate drop down list select the water rate

 

·    If you have selected Open Plot, perform the following steps:

       a. In the Usage drop down, select the appropriate from the usage drop down.

       b. In the Occupancy drop down, select who has occupied the Open plot.

                   c.  In the Open Plot Manual ALV, field, enter manual ALV related to plot.

                 d. In the Name of the Occupier, enter name of the land occupier.          

                           

·    If you have selected State Government, perform the following steps:

        a. In the Building Cost field, enter the total cost of the building.

·    If you have selected Central Government, perform the following steps:

        a. In the Building Cost field, enter the total cost of the building.

        b. In the Amenities section, select the appropriate from Amenities drop down list.

                                        

                                        Table 4: Floor Details

Field

Description

Floor Number

The number of the floor which is newly constructed. For example, if you have newly constructed the 3rd floor, you need to specify 3 in this field. For multiple floors, see the NOTE at the end of this table.

Type

Type can be: In which type the floor is constructing.

Name of the Occupier

In the Name of the Occupier, enter name of the land occupier.

Accessable Area

The total area in Sq.mtrs after deducting 15%

Usage

The new construction is used either for residential or commercial purpose.

Occupancy

The new construction is occupied by the owner or is given out for rent.

Structural Factor

From Structural factor drop down list select appropriate one.

Age factor

Select Age factor from the Age factor drop down list.

 

NOTE: If the property has multiple floors, enter the respective floor details. Click the Plus (+) icon and specify the details of each floor. Similarly, click the Cross (X) icon to delete a floor.

 

 

9.   Click Forward to Forward the application to high authorities.                

10.  Click Approve to Approve the changes 

11.  Click Close to exit from the Window.
