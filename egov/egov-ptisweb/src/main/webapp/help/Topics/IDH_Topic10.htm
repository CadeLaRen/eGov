Introduction

 

Every information about each property that comes under the jurisdiction of the Corporation has to be well maintained in the system.

 

A new property has to be created in the system when an assessor discovers it during the field survey or when a citizen informs the Corporation about the property. When a property is bifurcated into multiple parts, you need to update the details. When two or more properties are merged together, the details must be updated. You need to update the details when the ownership of a property is changed from one owner to another.
