Changing Property Address

 

Occasionally, there may be clerical errors in the address that you have typed while registering a property. You can correct such errors at the request of the property owner after receiving necessary proof.


To modify an address, perform the following steps:

1.   Find the property  which you want to change the address of.

2.   From the Actions drop-down list, select Change Property Address to display the Property Particulars page.

3.   You get the property address details that were entered while registering the property. You can make the necessary changes.

 

NOTE: You need to type a comment in the Comments field while modifying a property address.

            

4.   Click Forward to Forward the application to high authorities.                

5.   Click Approve to Approve the changes.

6.   Click Close to exit from the screen.
