Bifurcation of Property

 

An existing property can be divided into two or more parts.

 

For example, "A" is the original property and if it needs to be given away to two individuals, namely "B" and "C", then you must divide the property "A".

 

Dividing a property includes the following tasks:

1.   Deactivating the original property, which is automatically done by the system.

2.   Registering the individual properties one by one.

 

In case of Bifurcation of an Property, perform the following steps:

1.   Find the property  which you want to change the address of.

2.   From the Actions drop-down list, select Bifurcation to display the Property Particulars page.

3.   Enter the details which you want to change.

 

4.   Select the type of the property from the Property Type drop-down list, and perform the following steps:

·         Fields which are common in all Property type details:

·    If you have selected Open Plot, perform the following steps:

       a. In the Usage drop down, select the appropriate from the usage drop down.

       b. In the Occupancy drop down, select who has occupied the Open plot.

                   c.  In the Open Plot Manual ALV, field, enter manual ALV related to plot.

3.   Click Forward to Forward the application to high authorities.
