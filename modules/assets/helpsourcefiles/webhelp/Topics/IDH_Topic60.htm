Creating an Asset Category

 

To create an asset category, perform the following steps:

1. In the Asset Management menu, click Asset Category.

2. Click Create Asset Category to open the asset category page.

3. Type the details as described in Table 1.

4. Click one of the following options:

·        Save: To save the asset category you have created

·        Close: To close the page without saving the details

 

Table 1: Creating an asset Category

Field

Description

Category Code

Type the code assigned to the asset category in the Asset Category field.

Asset Category Name

Type the name of the asset category in the Asset Category Name.

Asset Category Type

Select one of the asset categories from the Asset Category Type drop-down list:

·        Land: If the asset is a land

·       Movable Asset: If it is a movable asset such as a vehicle or a piece of furniture

·       Immovable Asset: If it is an immovable asset such as a building or a bridge

Parent Category

An asset can be assigned to a specific parent asset, if it has been created as the subset of the latter. For instance, we  can say that 'Automobiles' is the parent category and 'Maruthi 800' is a sub category of it.

 

Select the parent category to which the asset belongs from the Parent Category drop-down list.

 

The items in this drop-down list appears depending upon the Asset Category Type you have selected.
