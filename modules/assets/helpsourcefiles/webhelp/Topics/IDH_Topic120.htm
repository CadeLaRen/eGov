Modifying an Asset Master

 

To modify an asset master, perform the following steps:

1. In the Asset Management menu, click Asset Master.

2. Click Modify Asset to open the search asset page.

3. Type the search details as described in Table 3.

4. Click one the following steps:

·                 Search: To find the asset master depending upon the search criteria

·                 Close: To close the page without performing the search

 

Depending on the search criteria, a list of assets is displayed.  You can see the following details of each of the assets:

·        Code

·        Name

·        Category

·        Department

·        Ward

·        Status

 

5. Click the Code link in the relevant row to open the edit asset page.

Make necessary modifications.

6. Click one of the following options:

·        Save: To save the changes that are made

·        Close: To close the page without saving the changes that are made
