Creating an Asset Master

 

To create an asset master, perform the following steps:

1. In the Asset Management menu, click Asset Master.

2. Click Create Asset to open the create asset page.

3. Type the details as described in Table 2.

4. Click one of the following options:

·                 Save: To save the asset master you have created

·                 Close: To close the page without saving the details

 

Table 2: Creating an Asset Master

Field

Description

Asset Code

Type the code assigned to the asset in the Asset Code field.

Name of Asset

Type the name of the asset in the Name of Asset field.

Department

From the Department drop-down list, select the name of the department that has created or purchased the asset.

Asset Category Type:

Select one of the asset categories from the Asset Category Type drop-down list:

·                 Land: If the asset is a land

·                Movable Asset: If it is a movable asset such as a vehicle or a piece of furniture

·                Immovable Asset: If it is an immovable asset such as a building or a bridge

Category

Select the category to which the asset belongs from the Category drop-down list.

Description

Type a short description of the asset in the Description field.

Select the area in which the asset is located from the Area drop-down list.
