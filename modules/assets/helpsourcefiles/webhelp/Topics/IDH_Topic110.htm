Viewing an Asset Master

 

To view the asset master you have created, perform the following steps:

1. In the Asset Management menu, click Asset Master.

2. Click View Asset to open the search asset page.

3. Type the details as described in Table 3.

4. Click one of the following steps:

·                 Search: To find the asset master depending upon the search criteria

·                 Close: To close the page without performing the search

 

Depending on the search criteria, a list of assets is displayed.  You can see the following details of each of the assets:

·        Code

·        Name

·        Category

·        Department

·        Ward

·        Status

 

5. Type the Code link in the relevant row to open the asset master page.

6. Click one of the following options:

·        Close: To close the page after viewing the details

·        Back: To go back to the previous page
