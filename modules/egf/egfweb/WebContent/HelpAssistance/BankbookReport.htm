BANK Book Report: 

Objective:

A Bank book is similar in all respects, to Cash Book, except for the fact that it records the following:

1. Bank transactions i.e. all cheque and cash remitted to bank

2. All cheques issued from the bank account

3. Transfers “to” and “from” the bank account are displayed.

4. The closing balance as at the “end date” is also displayed.

Field Description Table:

Field Name Field Type Field Description
Bank Drop Down List of Bank will be shown from which we can select .
Account number Drop Down Account numbers will be displayed.
Start date Date Date of beginning of the report to be generated.
End date Date To date till which the report needs to be generated.

Expected Results :

Action Expected Result
Search Receipts and Payments will be generated for that particular bank and account.
Print The report generated can be printed out out using this option.
Cancel Refreshes the screen .
Close Closes the screen.
