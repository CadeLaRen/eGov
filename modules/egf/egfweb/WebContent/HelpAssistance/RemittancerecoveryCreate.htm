Remittance Recovery Create: 

Objective:

The Recoveries can be remitted using - Create Remit Recovery screen. On search, based on the given criteria, the list of invoices will be shown for the selected Recovery for a given month and Fund. The voucher number and the bank details from which the payment is made should be entered to save the remittance. The amount can be amended if required in case where a partial payment is made.

Field Description Table:

Field Name Field Type Field Description
Fund Select List Select the Fund for which the invoice belong to and remittance has to be made.
Recovery Name Select List Select the Recovery type for which the remittance is made.
Month Select List Select the month for remittance. Month for which the invoices were created has to be searched.
Year Select List Select the year for which the remittance is made.

Expected Results :

Action Expected Result
Search Enter the search criteria as given above and click this button to list the invoices for remittance. The below detailed screen as given will be shown on click. The remittance amount against each invoice will be shown which can be amended if required before creating the remittance.
Cancel Cancels the transaction and loads the page afresh.

On save, the system will pass a Payment voucher: Dr. Recovery Code Cr. Bank If the Recovery code is designated as a control code, the sub-ledgers will also be updated.

Saving the Transaction :

Action Expected Result
Save & Close Submits the transaction. On successful submit, the system creates the remittance (Payment Voucher) and the screen is closed.
Save & New Submits the transaction. On successful submit, the system creates the the remittance (Payment Voucher) and the same screen is made available for entering a new record
Save & View Click this button to save the entry, create the remittance and view the print preview of the voucher and print it. The following screen shows the print preview of the voucher.
Show GL Entry Shows the entry that will be passed to the GL on submitting the transaction.
Cancel Cancels the transaction and loads the page afresh.
Close Close the screen
