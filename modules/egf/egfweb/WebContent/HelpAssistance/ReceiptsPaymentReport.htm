Receipt and Payment Report: 

Objective:

This is a schedule of all Receipts received and Payments made during a past period or for a particular financial year.

Field Description Table:

Field Name Field Type Field Description
Period Drop down Options  of yearly /half yearly / monthly are provided.
Year Drop Down Options  of years/months are provided.
Amount Drop down  Select the required option for display of amount in Rupees/Thousands/Lakhs

Expected Results :

Action Expected Result
Submit Receipt and Payment Report will be generated for that particular period.
Print The report generated can be printed using this option.
Cancel Refreshes the screen .
Close Closes the screen.
