Contract Type Modify: 

Objective:

This screen enables to modify the existing Contract types which have been created.

Work/Contract type master is used to define the types of contracts/works that can exist. For Ex: “Professional” and its sub types can be Professional - 1.Legal 2.Medical .All the W.O when created should be mapped to a particular work type, so that all the invoices for that WO can be linked to a defined recovery for that Work type.

Field Description Table:

Field Name Field Type Field Description
Code User Entry The user defined contract-type code
Parent code Select List The parent to which a sub type is mapped.
Applied To Select List The type of party to which the contract type is linked. This means, the defining type’s work order can only be created for the given party type.
Description Text  Enter any description for the type created.

Expected Results :

Action Expected Result
Search Enter the search criteria as given above and click this button to list the Contract types created which can be modified with required details.
Close Closes the screen.
