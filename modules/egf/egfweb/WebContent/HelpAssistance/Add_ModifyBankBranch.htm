Bank Enquiry / Modify : 

Objective: To create / modify a Bank account, the Bank Accounts Menu is selected from the Menu Tree, on clicking which, the Bank-Enquiry screen opens up.

Field Description Table:

Field Name Field Type Field Description
Bank Name VarChar Bank Name for which the details need to be modified

Expected Results:

Action Expected Result
Modify Screen to modify the existing Bank name will open.
Add Branch Screen to add a new branch for the existing Bank will open.
Close Closes the Bank Enquiry screen.
Modify Branch Upon clicking on the particular branch which is shown , the modify screen with options for Add branch, Add account, Modify accounts will open up.
