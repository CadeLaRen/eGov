Procurement Order Search : 

Objective: Screen to View Procurement Orders. The user selects one or more criteria and the system returns all the orders that match the criteria. The users can then click on a order number and see the details of the specific record..

Field Description Table:

Field Name Field Type Field Description
P.O number Search field Enter the P.O number to be modified or click the lens provided which displays the list and then select the one to be modified.
Type Drop down Supplier / Contractor
Supplier/Contractor Name Drop Down Select from the list
P.O date from Date Enter the date from which the record needs to be displayed.
P.O date To Date Enter the date till which the record needs to be displayed.

Expected Results :

Action Expected Result
Search Returns all the records in a tabular form which satisfy the search criteria. User can click on the order number and view the details of each. .
