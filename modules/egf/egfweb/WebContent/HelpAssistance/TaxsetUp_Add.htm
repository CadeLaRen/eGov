Tax Set Up: 

Objective: The Tax Set-up master is used to define a tax to an account code, and to set Cess percentages wherever required.

The ULB can levy multiple taxes and for each tax type, a pre-designated General Ledger code will need to be designated. In addition, the ULB may also need to levy a cess on the tax, typically on behalf of the state government. The cesses will be treated as liabilities that need to be paid out to the state government. In the current ULB environment, cesses are levied on Property Tax only.
