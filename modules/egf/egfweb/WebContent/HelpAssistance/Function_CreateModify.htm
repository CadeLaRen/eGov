Function : 

Objective:

Function code is taken for each account code being used (debited/ credited) in a transaction. This will be done at the line level in a transaction. This is an optional field with a default of 00. Budgets are drawn up and stored at a function level .Function codes can have a hierarchy of any number of levels. Only leaf-level functions (i.e. function with no sub-functions) will be used for transactions.
