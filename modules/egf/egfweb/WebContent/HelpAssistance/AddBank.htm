Bank Master : 

Objective: All bank accounts operated by the ULB need to be defined as master data. Bank accounts maintained by the ULB must satisfy the following conditions: Each bank account must belong to a specific fund because it is mandated that bank accounts must be maintained by fund. Each bank account must belong to an account type. Each bank account must have a specific General Ledger code. The GL code will be determined based on the account type.

Add Bank : 

Objective: This screen is for creating the Banks. For adding a bank, the “add Bank” button should be clicked, which opens up the “Bank-New” screen. By giving a bank name and saving, a new bank is created.

Field Description Table:

Field Name Field Type Field Description
Bank Name VarChar Bank Name to be added
Narration VarChar Any info about bank

Expected Results :

Action Expected Result
Save and close New Bank will be created
Add Branch Screen to create new branch will be opened
Cancel Cancels the entry made.
Close Screen is closed
 

Modify Bank : 

Objective: This screen is for editing the Bank name already created. For editing a bank name, the “modify Bank” button should be clicked, which opens up the “Bank-Modify” screen. By changing the  bank name and saving, the record gets modified.

Field Description Table:

Field Name Field Type Field Description
Bank Name VarChar Bank Name to be added
Narration VarChar Any info about bank

Expected Results :

Action Expected Result
Submit Bank Details will be successfully modified.
Close Screen is closed
