Objective: Supplier/Contractor master captures the master data related to a third-party providing goods or services to the ULB. Prior to raising a Work Order or a Purchase Order, the ULB must define the Supplier/Contractor in the masters.

Field Name Field Type Field Description

City Alphanumeric City Name
Pin Numeric Pin code
Mobile Alphanumeric mobile number for contact
Contact Person Alphanumeric any other contact person name
Bank Name Alphanumeric Bank Name where the supplier/contractor has his account
Bank Account Alphanumeric Account number
Disbursement Mode Drop Down Cheque/ECS
