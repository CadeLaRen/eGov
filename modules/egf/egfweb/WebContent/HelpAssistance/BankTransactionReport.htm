Bank Transaction Report: 

Objective:

This report will generate the details of all the bank transactions done with the particular bank for that particular period. The transaction type (Receipts/Payments) report  can also be generated.

Field Description Table:

Field Name Field Type Field Description
Bank Drop Down List of Banks will be shown from which we can select .
Account number Drop Down List of Account numbers for that bank will be displayed.
Transaction type Drop Down Receipts / Payments can be selected.
Starting date Date Date of beginning of the report to be generated.
End date Date To date till which the report needs to be generated.

Expected Results :

Action Expected Result
Search Bank Transaction report will be displayed for that particular fund and period.
Cancel Refreshes the screen .
Close Closes the screen.
Print Preview The report generated can be previewed before printing it out ,using this option.
