Contractor/Supplier Report: 

Objective:

This report gives the details of each work taken up by the Contractor or Supplier  .

Field Description Table:

Field Name Field Type Field Description
Type Drop Down Contractor or Supplier option to be selected.
Code Search Lens Code of the Contractor/Supplier.
Name Alphabetic Name of Contractor or Supplier will be displayed.

Expected Results :

Action Expected Result
Submit Report will be generated for that particular Contractor /Supplier.
Cancel Refreshes the screen .
Close Closes the screen.
