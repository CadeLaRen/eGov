Procurement Order : 

Objective: Prior to passing a supplier or contractor bill, it is required to have a Supplier/Contractor Master defined and a Procurement Order  (Purchase Order/Work Order respectively) defined in the system.

Field Description Table:

Field Name Field Type Field Description
Type Mandatory – check box Supplier / Contractor
Order No Mandatory-alphanumeric field Order code for identification of bill.
Name of work Mandatory-alphanumeric field Name to be given for that work.
Order Date Mandatory - Date field Date of the order.
Total Value Mandatory-Numeric Total money value of that order.
Advance Payable Optional-Numeric Any advance amount to be paid for this order has to be mentioned.
Authorized by Optional - Text Name of the person who authorizes the order.
Level of Work To select between “City”, “Zone” and “Ward” Select the option from the list of drop down.
Work Type Drop Down To select from the drop-down list
Sub Type Drop Down To select from the drop-down list
Ordered to Drop Down To select from the drop-down list of Suppliers/Contractors already created
Type of Work Drop Down Mandatory only if ordered to Contractors Capital Works Repaid Works Other Service
Fund Name Drop Down To select from the drop-down list of Funds already created
Financing source Drop Down To select from the drop-down list of Financing Sources already created
Retention Mandatory-alphanumeric field  
Security Deposit Optional-Numeric  
PAN No Optional-alpha Numeric  
TIN No Optional- alpha Numeric  
Status Check Box Only Active Procurement Orders are available for bills
Sanction no Optional-alphanumeric  
Sanction Date Optional-Date  
Sanction no Optional-alphanumeric  
Narration Optional-Text  
Remarks Optional-Text  

Expected Results :
 
Action Expected Result
Save and New Saves the record and opens another screen for new procurement order creation.
Save and Close Saves the record and closes the screen.
Cancel Refreshes the screen.
Close Closes the screen.
