Expected Results and/or Workflow:
This Report having two sections, Receipts and Payments.

Action Expeced Result
Create a cash or cheque receipt entry, and check the report This entry should reflect in the Receipts section of the report
Create a payment entry, and check the report This entry should reflect in the Payments section of the report
