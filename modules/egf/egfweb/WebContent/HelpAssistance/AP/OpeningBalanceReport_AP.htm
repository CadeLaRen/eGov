Objective: It will give the opening balances for the selected Financial Year.

Field Description Table:

FieldName FieldType FieldDescription
Fund Text, optional Drop down : list of funds
Financial Year Alphanumeric Drop down : list of all financial year
Fund Text Selected fund name, if fund is selected
Glcode Number Account code
Account Head Text Account code description
Description Text Opening Balance Description
Debit (Rs.) Number Opening Balance amount, if it is debit
Credit (Rs.) Number Opening Balance amount, if it is credit

Expected Results and/or Workflow:
Opening Balance can either be a debit or credit balance.

Action Expeced Result
Check the report The report should show the difference between the opening debit and credit balances for each fund.
Check the report The report should show the Total for each Fund and the grand total across all funds.

Troubleshooting:
Question1:
If you didn't select the fund in the input section
Answer1:
The report will take time to get it, because it will show all the funds
