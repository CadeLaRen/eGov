Objective: This report will list out the ledger entries of the transaction for that date range for the selected account code.

Field Description Table:

FieldName FieldType FieldDescription
Account Code Text, required select from the search screen
Fund Text, required Drop down : list of funds
Financing Source Text, Optional Drop down : list of Financing Source
Starting Date Date, required Starting Date of the report
End Date Date, required End Date of the report
Voucher Date Date Voucher Date
Voucher# Alphanumeric Voucher number
Particulars Text All entries for that voucher except the account code for which the ledger report is being taken. There can be debit and credit entries - the debit/credit entries is be marked with Dr/Cr
Amount (Rs.) Number amount for the glcode for which the report is being taken

Expected Results and/or Workflow:
The report having facility on clicking the voucher number, the user can able to drill down to the individual voucher.

Action Expeced Result
Create a transaction voucher and check the report The voucher entry will reflect in the report.
Chec the report It will show the opening and closing balance of that selected account code.
