Objective: It will give a list of all vouchers created for that date range (across funds).

Field Description Table:

FieldName FieldType FieldDescription
Starting Date Date Starting Date of the report
End Date Date End Date of the report
Fund Text Drop down: list of funds
Date Date Voucher Date
Voucher# Text Voucher number
Type Text Voucher type, i.e. Pay In Slip - Contra, Other receipts - Receipt, JVGeneral - Journal Voucher, Salary Journal - Journal Voucher...
Narration Text Description of the voucher
Status Text Voucher status like Unconfirmed, Confirmed, Reversal, Reversed
Glcode Text Account code, which is going to credited or debited
Particulars Text Account code description
DrAmt (Rs.) Number Debit amount
CrAmt (Rs.) Number Credit amount

Expected Results and/or Workflow:
All the voucher will list out here for that date range. create a voucher and check the report. The report having facility on clicking the voucher number, the user can able to drill down to the individual voucher.

Action Expeced Result
Create a voucher and check the report In the report, the status should be unconfirmed and check the other entries also
Approve the voucher Status should be confirmed.
