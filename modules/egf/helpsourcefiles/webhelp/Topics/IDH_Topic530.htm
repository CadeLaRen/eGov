Creating Accounting Entities

 

An accounting entity is a record of financial transactions and events in the book of accounts.

 

To create an accounting entity, perform the following steps:

 

1. Click Masters to display the sub-menu tree.

2. Click Scheme > Accounting Entity to display the Accounting Entity Create page.

3. Enter the details as described in the table.

4. Click one of the following options:

·         Add to save the changes.

·         Close to close the Window without saving the changes.

 

Table: Create Accounting Entity

 

Field

Description

Name

Type the name of the entity. For example, if you create an entity to track the loan that was given to an employee, you can type "loan" in the Name field.

Description

A brief description about the entity that you create.
