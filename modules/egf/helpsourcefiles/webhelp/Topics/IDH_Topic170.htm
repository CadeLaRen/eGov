Reappropriating a Budget

 

To reappropriate a budget, perform the following steps:

1. Expand the Budgeting folder to show its subfolders.

2. Click the Add/Release link under Budget Reappropriation. You get the Add Budget Details page.

3. Select the financial year - for which the budget was defined - from the Financial Year drop-down list.

4. Select the department which executes the projects from the Executing Department drop-down list.

5. In the Add/Release from existing budget line item area, select a budget group whose values you want to change.

6. Select a function from the Function drop-down list.

7. To get the actuals for the selected group as on a specific date, type a date in the Date field and click Get Actuals. The following details are displayed:

·      Sanctioned amount for the selected budget group

·      Expenditure incurred as on the specified date for the selected budget group

·      Balance amount that is available

8. Do one of the following steps:

·      To allocate more fund for the selected line item, select Addition from the Change Requested drop-down list and type an amount in the Addition of Fund Sought field.

·      To deduct fund from the selected line item, select Deduction from the Change Requested drop-down list and type an amount in the Addition of Fund Sought field.

9. To add new line item to a budget group, in the Add new budget line item area:

·      Select the budget from the Budget drop-down list.

·      Select the budget group from the Budget Group drop-down list.

·      Select a function from the Function drop-down list.

10. Click Save to save the changes.

 

NOTE: On saving the changes, a reappropriation number will be assigned to the budget, and the budget will go for approval by the concerned officials.
