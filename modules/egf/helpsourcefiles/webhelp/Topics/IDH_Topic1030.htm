Creating Collection/Payment Points

 

This is used to identify the Collection Points (receipts) and payment points (payments) for tracking and audit. 

 

To create it, perform the following steps:

 

1. Click Masters > Collection/Payment Point to display a new page.

2. Enter the values as described in the table.

3. Click one of the following options:

·      Save and New to save the changes and create another supplier/contractor.

·      Save and Close to save the changes and close the window.

 

 

Field          

  Description

Code

The code provided to the collection/payment point

Name

The name of the collection/payment point

Department Code

List of departments

Department Name

From the department code selected

Entity Type

Valid list

·         Bill Collector

·         Counter

·         Cashier

Narration

The description of the collection/payment point

Active

Default to Y
