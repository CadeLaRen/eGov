Creating/Modifying Source of Finance

 

Source of Financing refers to a financing source that contributed to a particular transaction. For instance, a work may be performed by funds provided by the World Bank – in which case, the Source of Financing is ‘World Bank Fund’. Source of Financing codes can have a hierarchy of any number of levels. Only leaf-level source-of-financing will be used for transactions.

 

To create a source of finance, perform the following steps:

 

1. Click Masters > Source of Finance to display the page which lists the current sources of finance.

2. Do one of the following steps:

·      To add a new source of finance, click New.

·      To modify a source of finance, click the radio button against the one that you want to modify, and click Modify.

3. Enter the details as described in the table.

4. Click one of the following options:

·      Save & New to save the changes and create another source of finance.

·      Save & Close to save the changes and close the window.

·      Cancel to clear the fields and close the window.

·      Close to close the window.

 

Table: Create/Modify Source of Finance

Field

Description

Financing Source Code

Type a numerical code to uniquely identify the source.

Financing Source Name

Name of the source that you want to create.

Parent Type

Select a parent type from the drop-down list.

NOTE: You need to select a parent type only if you want to create a child financing source under it.

Parent Code

The Parent code automatically appears once the parent Type is selected.

Active

By selecting this check box, the financing source becomes active, and accounting of transactions under the source is enabled.
