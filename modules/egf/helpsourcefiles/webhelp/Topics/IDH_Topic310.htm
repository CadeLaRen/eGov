Creating Expense Bills

 

Expense bills are used to cover miscellaneous unplanned payments that are not based on any purchase or work order.

 

To create a C Bill, perform the following steps:

1. Click Transactions> Expenditure > Bill Registers> Create Expense Bill to display the Expense Bill-Create page.

2. In the Header area, enter the details as described in the Table 1.

3. In the Credit and Debit section, enter the details as described in Table 2.

4. In the Approval Information section, enter the details as described in Table 3.

 

NOTE: Using the eGov Admin module, the application enables you to employee either a manual workflow or an automatic workflow that suits best for your business requirements. If you have chosen to use a manual workflow, while creating the bill, you need to select the department and designation of the person who needs to approve it. When you use an automatic workflow, you do not need to manually select these fields. The system automatically forwards the bill voucher to the relevant users for approval.

 

5. Click Save and Forward to save the changes and send the details to the officer you have selected for approval.

 

NOTE: Depending on the bill sub-type that you have selected, the system generates a check list. You need to furnish "Yes/No/NA" answer to each of the items in the check list depending on its relevance.

 

The bill will be available in the Inbox of the concerned officer for approval and voucher generation.

 

The following rule is applied to calculate the amount to be credited to this account:

 

Sum (Debit Accounts) – Sum (Credit for deductions, adjustments etc)

 

The following validations are performed in addition to the standard check of (Debits=Credits):

Net Payable = Passed Amount – Adjustment Amount – Sum (deductions)

Functionary

Narration

Any comment that you want to make

Sub-Ledger Type

Select the applicable sub-ledger type from the drop-down list

Select the applicable bill sub-type from the drop-down list. Depending on the bill type you have selected, a check list is displayed.

Creditor Code

Select a creditor code, which depends on the sub-ledger type you have selected.

Party Bill Number

The bill number of the party for whom the bill is created

Party Bill Date

Date when the party created the bill

Pay To

Name of the party to whom the payment is to be made

Table 3: Approval Information

Field Name

Field Description

Approver Department

Select the department to which the officer who needs to approve the bill belongs.

A list of designations pertaining to the selected department is displayed. The list will include only those designations that belong to the next level of hierarchy. You can select a designation from the drop-down list.

Approver
