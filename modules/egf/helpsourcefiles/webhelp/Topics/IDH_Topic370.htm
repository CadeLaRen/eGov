Viewing a Procurement Order

 

You can view the procurement order that you have previously created.

 

To view a procurement order, perform the following steps:

1. Click Transactions to display the sub-menu tree.

2. Click Expenditure > Procurement Orders > View Procurement Orders to display the Procurement Order Create page.

3. Click one of the following options:

· Contractor to view an order created for a contractor.

·      Supplier to view an order created for a supplier.

4. Enter the values as described in the Table.

5. Click Search to display the PO.

 

 

 Table : Procurement Order Field Descriptions

FIELD NAME

DESCRIPTION

Order Number

Order number of the procurement

Order Date

Date when the procurement order is created

Name of Work

Name the project. Eg: Construction of a flyover

Total Value

Total estimated value of the project

Advance Payable Amount

The amount that is intended to be paid as advance

Authorized By

Name of the person who did the authorization to make the payment

Level of Work

Select the jurisdictional area where the work is executed. Eg: Zone, Ward, City.

Work Type

Select the type of work. Eg: Roads, Bridges.

Sub Type

Sub type of the project. Eg: Roadside drain.

Type of Work

Select the type of the work. Eg: Capital Works

Ordered To

Name of the contractor to whom the project is assigned for execution

Department

Name of the department which executes the project

Fund Name

Name of the fund which is utilized to execute the project

Financing Source

Select the source of finance from the drop-down list.

Scheme

Select a scheme from the drop-down list. The scheme names are displayed based on the fund selected.

Sub Scheme

Select a sub scheme from the drop-down list. The sub schemes are displayed based on the scheme selected.

Security Deposit

The amount that was deposited at the ULB as security deposit by the contractor.

Sanction Number

Type the sanction number

Sanction Date

Date when the project is sanctioned

Narration

Description about the project which gets executed

Fixed Asset

Select this check box if the project that gets executed is a fixed asset.

Status

 

Remarks
