Day Book Report

 

A Day Book lists out all transactions of the ULB for the selected period, irrespective of whether or not they are cash/bank transactions.  The day book is more or less in the same format as a journal book, with the difference that it displays cash/bank transactions also, whereas the journal book displays only non-cash/bank transactions.

 

To generate a Day Book Report, perform the following steps:

1. Click Reports > Financial Statements > Day Book.

2. Enter the details as described in Table 1.

3. Click Submit to generate the report.

You can export the report in both Excel and PDF formats.

 

Table 1: Day Book Report

Field

Description

Starting Date*

Starting date from when you want to generate the report.

End Date*

Date till when you want to generate the report.

Fund

Name of the fund.
