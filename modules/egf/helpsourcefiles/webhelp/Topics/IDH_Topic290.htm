Creating Work Bills

 

Works bills are created when you need to make a payment to contractors on completion or partial completion of the work that was assigned to them.

 

NOTE: You need to create a works bill in Financials ONLY if you have not implemented eGov Works module.

 

To create a works bill, perform the following steps:

1. Click Transactions> Expenditure > Bill Registers> Create Works Bill to display the Works Bill-Create page.

2. Enter the details as described in the table 1.

3. Click one of the following options:

·      Save and New to save the changes and create another works bill

·      Save and Close to save the changes and close the window

·      Close to close the Window without saving the changes

 

The system automatically generates a bill number using which you can track the bill at a later point in time. The bill will be available to the concerned official for approval and voucher generation.

 

NOTE: The following rule is applied to calculate the amount to be credited to this account:

Sum (Debit Accounts) – Sum (Credit for deductions, adjustments etc)

 

The following validations are performed in addition to the standard check of (Debits=Credits):

Bill amount  >= Passed amount + Adjustment amount 

Debit Total  =  Passed amount

Net Payable = Passed Amount – Adjustment Amount – Deductions (calculated)

 

Table 1: Create Works Bill

Field Name

Description

Bill Date

Date of creation of the bill

Expenditure Type

It is displayed automatically as "works"

Bill Type

Specify whether the bill is a final bill or running bill.

WO/PO Number

Work order number. Click the button to select an order. Name of the contractor and the total work amount are automatically displayed.

Contractor

Name of the contractor is displayed after entering the work order number.

Department

Select the name of the department which executes the work.

Total Work Order Amount

Amount agreed upon by you and the contractor to execute a work.

Total Billed Amount

The total amount that has been paid so far towards the work order. This is automatically filled in.

Advance Not Yet Adjusted

This field is applicable only when an advance was paid to the contractor.

Advance Adjusted

This field is applicable only when an advance was paid to the contractor.

Bill Amount

This amount that needs to be billed for.

Passed Amount

The amount that has been approved to be paid.

MB Ref No

Reference Page Numbers of the Measurement Book.

Functionary

Name of the functionary

Narration

Any comment that you want to include.
