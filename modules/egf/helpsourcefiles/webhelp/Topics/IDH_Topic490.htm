Creating a Bank

 

Whenever you want to operate with a new bank/branch, you need to create it in the system. In addition, for each bank account, the list of cheques issued by the bank has to be assigned. On all payment screens, the system checks for the validity of the cheque number - i.e. it has to be within the specified range and it must be available.

 

To create a bank, perform the following steps:

 

1. Click Masters to display the sub-menu tree.

2. Click Chart of Accounts > Add Bank to display the Add Bank page, where the already added bank names are listed.

3. Type the name of the bank you want to add in the Bank Name field.

4. Type a description about the bank in the Narration field.

5. Click Add Branch to save the name of the bank and to display another page where you can provide the branch details.

6. Enter the details as described in the tables.

7. Click one of the following options:

·      Save and New to save the information and add another branch details

·      Save and Close to save the information and close the window.

Select one of the following options:

·         Receipts: if use the bank to operate only on receipts

·         Payment: if use the bank to operate only on payments

·         Receipts and Payment: if you use the bank to operate on both receipts and payment accounts

Narration

Any description about the bank.

Active

Select this check box to make the bank operational. Transactions can be done through only active banks.
