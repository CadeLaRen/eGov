Concurrence Report

 

Before a cheque is assigned to any party, the person who issues the cheque needs to perform a bank balance check. Using the Concurrent Report, the person can figure out the payments for which the cheque can be issued.

 

This report throws light on the financial status and the liquidity of the ULB thereby enabling you to take a fair decision on the dispersal of cash to the creditors.

 

To generate a concurrence report, perform the following steps:

1. Click Transactions > Expenditure > Payments > Concurrent Reports. You get the Concurrence Report page, which displays the following search criteria to generate a report:

·         Name of the Bank

·         Account Number

·         As On Date

·         Current Receipts

 

2. Click Submit to display the report according to the given search criteria.

 

The system lists all the BPVs that are approved but cheques not assigned along with the values selected as the header information.

 

 

Table: Concurrent Report

Field Name

Description

Name of the Bank

Name of the bank through which payment is done

Account Number

Account number that you hold with the bank

As On Date

Date as on which you want to generate the report

Current Receipts

This is the receipt that is not present in the system for the selected bank account
