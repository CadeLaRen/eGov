Modifying a Receipt

 

This feature is used to make any modifications to a receipt.

 

1. Click Transactions > Receipt > Modify Receipts to display the Modify Receipt search page, which displays the following search criteria to find a receipt:

·      Receipt Type

·      Voucher Date From

·      Voucher Date To

·      Department

2. Click Search to list out the voucher according to the given search criteria (see table).

3. Select the voucher to be modified, and the voucher details are displayed.

You can modify the values to suit your requirements.

4. Click Save.

You get a message indicating that the voucher has been successfully modified.
