Modifying a Salary Bill Voucher

 

Sometimes, you may need to modify the salary bills that you have previously created.

 

To modify a salary bill voucher, perform the following steps:

 

1. Click Transactions> Expenditure > Bills Accounting> Modify Bills to display the Modify Bill Search page, displaying the following search criteria to find a voucher:

·      Bill Type

·      Voucher Number

·      Fund associated with the transaction recorded in the voucher

·      Voucher Date From

·      Voucher Date To

·      Department

 

2. Enter the details as above and click Search to list the vouchers (see table below).

3. Click on the voucher number to display the voucher details. You can make any modification to the voucher to suit your requirement.

4. Click Save and Close to save the changes and close the Window.

 

A new voucher number is displayed on the screen.

 

NOTE: Only vouchers of "Unconfirmed" status can be modified.

 

Table: Modify Salary Bill Voucher

Field

Description

Bill Type

The type of the bill (salary, purchase, C Bill, works)

Voucher Number

Voucher number corresponding to the bill.

Fund

Fund associated with the transaction

Voucher Date From

Start date of the voucher. For example, to view a bill voucher that was created after Jan 1, 2010, then type 01/01/2010.

Voucher Date To

End date of the voucher. 

Department

Enter the name of the department which created the bill
