General Ledger Report

 

The General Ledger classifies the transactions of a ULB for a specified period under different detailed account codes. It also gives the net balance in the account for the selected period.

 

To generate a General Ledger Report, perform the following steps:

1. Click Reports > Financial Statements > General Ledger.

2. Enter the details as described in Table 1.

3. Click Submit to generate the report.

You can export the report in both Excel and PDF formats.

 

Table 1: General Ledger Report

Field

Description

Account Code*

Select an account code from the drop-down list.

Fund

Name of the fund.

Financing Source

Name of the financing source.

Starting Date*

Date starting when you need to generate the report.

End Date*

Date till when you need to generate the report.

Department

Select the name of the department from the Department drop-down list.

Functionary

Name of the functionary.

Function

Name of the function.

Field

Ward number.
