Functionwise Budget Summary

 

This report lists out functionwise budgeted amount across all departments.

 

To generate a functionwise budget summary across all departments,

perform the following steps:

1. In the Reports menu, click Budget Reports > Functionwise Budget.

You get the Budget Report- Departmentwise page, where you have various search criteria to generate a report.

2. From the Financial Year drop-down list, select the financial year for which the budget was defined.

3. From the Department drop-down list, select the name of the department for which the budget was defined.

4. From the Type drop-down list, select the type of the budget (Revenue or Expense).

5. From the Function drop-down list, select the name of the function to which the budget is associated.

6. Click Print to generate the report.
