Creating Salary Codes

 

Each time you create a salary bill for an employee, you need to specify the earning and deduction salary heads, such as basic pay and TDS. If you create masters for all possible earnings and deductions, these items will automatically be displayed on the Create Salary Bill page.

 

This screen is used for setting up the earnings and deduction codes that are used in Salary bill. Each salary code is tied to a specific ledger code and this is used as a template for creating Salary bills.

 

To create salary codes, perform the following steps:

1. Click Set-up > Salary Codes > Create Salary Codes to display the Salary Codes Create page.

2. Enter the details as described in the table:

 

Table: Create Salary Code

Field Name

Description

Head

Name of the pay head. Example: Basic Pay

Account Code

Select the account code from the chart of accounts

Account Description

This field is automatically displayed as it appears in chart of accounts

Type

Select one of the following options:

·             Earning - if the pay head is an earning component such as HRA (House Rent Allowance)

·             Deduction- if the pay head is a deduction component such as PF (Provident Fund)
