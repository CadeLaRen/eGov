Creating Salary Bills

 

Whenever you are required to make a salary payment to an employee, you need to create a salary bill.

 

To create a salary bill, perform the following steps:

 

1. Click Transactions> Expenditure > Bill Registers> Salary Bills > Create Salary Bill to display the Salary Bill-Create page, where you have the following sections:

·      Header: Details such as financial year, month, and department for which you need to create a salary bill.

·      Earning: The earning payhead details such as basic pay and daily allowance (DA).

·      Deduction: The deduction payhead details such as TDS, Professional Tax, and PF.

 

2. Enter the header details as described in table 1.

3. Enter the earnings as described in table 2.

4. Enter the deductions as described in table 3.

3. Click one of the following options:

·      Save and New to save the changes and create another salary bill

·      Save and Close to save the changes and close the window

·      Close to close the Window without saving the changes.
