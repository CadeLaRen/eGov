Modifying Budget Definitions

 

You may need to modify an active budget that is defined previously to include some changes.

 

To modify a budget definition, perform the following steps:

1. Expand the Budgeting folder to list its subfolders.

2. Expand the Budget Definition folder to show the following hyperlinks:

·      Create Budget Definition

·      Modify Budget Definition

·      View Budget Definition

3. Click Modify Budget Definition to show the Budget Definition Search page.

4. Select the financial year for which the budget that you want to modify was defined from the Financial Year drop-down list.

5. Click Search to list the budgets for the selected financial year.

6. Click the name of the budget that appears in the Name column to show the budget definition.

7. You can modify the definition to suit your requirements. For field descriptions, see Creating a Budget Definition.

8. Click Save to save the changes.
