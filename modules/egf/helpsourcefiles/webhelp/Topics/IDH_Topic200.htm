Modifying a Reappropriated Budget

 

You can later on modify the values specified in a budget that has already been reappropriated.

 

To modify these values, perform the following steps:

1. Expand the Budgeting folder to show its subfolders.

2. Click the Modify link under Budget Reappropriation. You get the Modify Budget Reappropriation page which lists out various criteria to find the budget you want to modify.

3. Click Search after specifying the search criteria to display the budget.

4. Modify the fields as necessary and click Save.
