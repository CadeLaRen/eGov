Bank Transaction Report

 

This report provides information on all transactions that are made by the ULB in a specific bank for over a period of time.

 

To generate the Bank Transaction report, perform the following steps:

1.   Click Reports > MIS-Bank Transaction

2.   Specify the details as mentioned in the following table:

 

Field

Description

Bank name

Select the name of the bank through which the transaction is done from the drop-down list.

Transaction Type

Select the type of transaction, either Receipt or Payments, from the drop-down list.

Account Number

Select the bank account number through which the transaction is done from the drop-down list.

Starting Date

Click the Calendar icon to select a date or type in the date from when you want to generate the report.

End Date

Click the Calendar icon to select a date or type in the date till when you want to generate the report.

 

3.   Click Search to generate the report.

 

 

NOTE: You can export the report in both Excel and PDF formats.
