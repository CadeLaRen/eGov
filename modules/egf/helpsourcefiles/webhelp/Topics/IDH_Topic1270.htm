Journal Book Report

 

A journal book lists out all journal entries passed for a Fund for a specified period. The journal entries include contractors’ bills, suppliers’ bills, and salary bills, i.e., all the entries which do not involve either cash account or a bank account.

 

To generate a Journal Book Report, perform the following steps:

1. Click Reports > Financial Statements > Journal Book.

2. Enter the details as described in Table 1.

3. Click Submit to generate the report.

You can export the report in both Excel and PDF formats.

 

Table 1: Journal Book Report

Field

Description

Starting Date*

Date starting from when you want to generate the report.

End Date*

Date till when when you want to generate the report.

Fund*

Name of the fund.

Financing Source

Name of the financing source.
