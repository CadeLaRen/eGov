Creating a Bill Voucher

 

You need to create a bill voucher for every bill that is approved.

 

To create a bill voucher, perform the following steps:

 

1. Log on to the application as an officer who has the privilege to create a bill voucher (for instance, accounts assistant).

2. Click Transactions > Expenditure > Bills Accounting > Create Bill Voucher. You get the Bill Voucher Create page, displaying the following fields:

·      Bill Type

·      Department

·      From Date

·      To Date

3. Click Search to display all bills according to the given search criteria.

4. Click the bill number for which you want to generate a voucher. The bill details are displayed.

5. In the Approval Information section, enter the information as described in Table 2.

This section will be available only if you choose to use a manual workflow.

6. Click Send for Approval for sending the voucher for approval.

 

The voucher will be available in the Inbox of the approving officer.

 

NOTE: Using the eGov Admin module, you can set up either a manual or an automatic workflow that suits best for your business requirements. If you have chosen to use a manual workflow, while creating the bill, you need to select the department and designation of the person who needs to approve it. When you use an automatic workflow, you do not need to manually select these fields. The system automatically forwards the bill voucher to the relevant users for approval.

 

 

Table 1: Create Bill Vouchers

Field Name

Field Description

Bill Type

Select a bill type from the drop-down list. Example: Purchase Bill

Department

Select a department that created the bill.

From Date

Type the starting bill date

 To Date

Type the ending bill date

Bill Number

Type the bill number

 

 

Table 2: Approval Information

Field Name

Field Description

Approver Department

Select the department to which the officer who needs to approve the bill belongs.

Approver Designation

A list of designations pertaining to the selected department is displayed. The list will include only those designations that belong to the next level of hierarchy. You can select a designation from the drop-down list.

Approver

Based on the designation you have selected, a list of users who are mapped to the designation is displayed. You can select a user name from the drop-down list.

Comments

Type any comment that you want to include.
