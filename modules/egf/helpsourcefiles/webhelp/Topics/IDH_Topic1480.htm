Creating Non-Bill BPVs

 

This feature is used to generate a BPV for all Works, Salary, and Inventory vouchers created using JV.

 

To create a non-bill payment BPV, perform the following steps:

 

1. Click Transactions > Expenditure > Payments > Non-Bill Payment.  You get the Non-Bill Payment Search page which displays the following search criteria to find a voucher:

 

·      Voucher Number

·      Voucher Type

·      Voucher Name

·      Voucher Date From

·      Voucher Date To

·      Fund

·      Department

 

2. Enter values in the relevant fields and click Search to list out the vouchers that meet the given search criteria. The list includes the following information:

·         Voucher Number

·         Voucher Type

·         Name of the journal

·         Voucher Date

·         Fund Name

·         Amount

·         Status of the voucher

 

3. Click the voucher number whose BPV needs to be generated. You get the Create Direct Bank Payment page.

4. In the Header section, enter the following information:

·         Name of the Bank in the Bank field.

·         Account Number

·         Name of the person to whom the payment is made in the Paid To field

·         Document Number

·         Document Date

 

4. In the Approval Information section, enter the information as described in Table 2.

This section will be available only if you choose to use a manual workflow.

 

5. Click Save and Forward to send the payment voucher to the Inbox of the specified officer for approval. A message appears on your screen indicating the designation and user name of the officer to whom the BPV has been forwarded.
