Creating a Financial Year

 

Financial Year is a key master setup that defines the financial accounting cycles for a ULB.

 

To create a financial year, perform the following steps:

1. Click Period-End Activities > Financial Year.

You get the Financial Year Setup page.

2. Click New to display the Financial Year Create/Modify page.

3. Enter the details as described in the Table.

4. Click one of the following options:

·      Save and Close to save the details and close the window.

·      Save and New to save the details and create another financial year.

NOTES: Financial Years can be created only in a forward sequence. In other words, 2009-10 cannot be created once 2010-11 has been created. There must be no overlap between the date ranges of financial years. There must be no overlap between the fiscal periods within a financial year.
