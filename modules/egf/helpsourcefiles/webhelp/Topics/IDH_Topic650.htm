Pay In Slip

 

Every day, ULBs receive thousands of cheque payments against taxes, fines, and various other services they offer to the public. Using the Pay In Slip feature, you can remit all such cheques to a specific bank under a fund.

 

To create a pay-in slip, perform the following steps:

 

1. Click Transactions > Contra Entries > Pay In to display the Pay In page.

2. Enter the details as described in Table 1.

3. Click Search to display the list.

4. Enter the details as described in Table 2.

 

 

Table 1: Pay In Slip

Description

Amount

The amount that you have deposited in the bank

Fund

The name of the fund is automatically displayed, depending on the bank you have selected.

Department

Select the name of the department associated with the transaction.

Scheme

Select the name of the scheme associated with the cash deposit.

Subscheme

Select a subscheme if any.

Functionary

Select a functionary from the drop-down list.

Fund Source

List of all the fund sources of the undeposited cheques

Field

Select a ward number associated with the fund. Example: Property Tax for Ward 10.
