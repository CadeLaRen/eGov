Creating Contract Types

 

All contractors in a ULB may not do all kinds of projects. For example, some contractors may only do works related to buildings. Therefore, you need to create a contract type to uniquely identify the contractors who perform some specific kind of work.

 

They types that you create will appear as a drop-down list in the Sub Type field of the Supplier/Contractor Create page.

 

To create a contractor type, perform the following steps:

1. Click Set-up > Contract Types > Create Contract Type to display the Contract Type Create page.

2. Enter the details as described in the table.

3. Click Save to save the information.

 

Table: Create Contract Type

Field Name

Description

Code

Enter a code to uniquely identify a contract type. For example, to identify a contract associated with Roads and Flyovers, you can create a contract type "RoadsnFlyover"

Parent Code

Select the parent code from the drop-down list.

Applied To

Select an option from the drop-down list.

Description

Type a brief description about the party type you have created.
