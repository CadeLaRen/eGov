Creating ULB Details

 

Before using the application, you need to enter the ULB information such as name, address, telephone number.

 

To create/modify the ULB details, perform the following steps:

1. Click Set-up > ULB Details to display a page which displays the ULB details.

2. Click one of the following options:

·      To modify the details, click Modify.

·      To close the page, click Close.

3. Enter the details as described in the table.

4. Click one of the following:

·         Submit to save the changes

·         Cancel to close the Window without saving the changes

·         Close to close the Window
