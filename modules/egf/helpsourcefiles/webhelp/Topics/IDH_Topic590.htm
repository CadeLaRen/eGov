Modifying a Journal Voucher

 

Sometimes, you may need to change the entries you have made in the JV before it goes for approval. For example, you realize that you have made a wrong entry in a JV which needs to be corrected.

 

To modify a JV, perform the following steps:

 

1. Click Transactions > Journal Vouchers > Modify Journal Voucher.

You get the Journal Voucher Modify page, where you have the following search criteria:

·      Voucher Number

·      Voucher Date From

·      Voucher Date To

·      Fund

·      Department

2. Click Search after providing the above search criteria. You get a list of vouchers.

3. Click the voucher number that you want to modify. You get the Journal Voucher Modify page.

4. Make the changes as necessary. For field descriptions, see Table 1, Table 2, and Table 3 of Creating Journal Voucher.

5. Click one of the following options:

·      Save and View to save the changes and view the modified JV

·      Save and Close to save the changes and close the window.

·      Close to close the Windows without saving the changes.

 

Table: Modify Journal Voucher

Field

Description

Voucher Number

Voucher number corresponding to the bill.

Fund

Fund associated with the transaction

Voucher Date From

Start date of the voucher. For example, to view a bill voucher that was created after Jan 1, 2010, then type 01/01/2010.

Voucher Date To

End date of the voucher. 

Department

Enter the name of the department which created the bill
