Viewing Remittance Recovery

 

You can view the recovery type that you have previously created at any point in time.

 

To view details regarding a specific remittance, perform the following steps:

1. Click Deductions > Remittance Recovery > View Remittance Recovery to display the Remittance Recovery View page. This page includes various search criteria to find a remittance.

2. Enter the values as described in the table.

 

 

Field Name

Description

Fund

Name of the fund from which the remittance to be paid.

Recovery Name

Name of the recovery. Example:  CPF, PF, PT

Month

Name of the month against which the remittance is made. For example, to make a PF remittance for an employee for January, select "January" from the drop-down list.

Year

Select an year against which the remittance is made.
