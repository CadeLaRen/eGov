Assigning a Cheque

 

Cheques can be assigned against all approved BPVs.

 

To assign a check, perform the following steps:

1. Log on to the application as an accounts assistant.

2. Click Transactions > Expenditure > Payments > Cheque Assignments. You get the Check Assignments Search page which includes various search criteria to find the payment against which you want to assign a cheque.

3. Enter the following information in the relevant fields:

·      Mode of Payment (cheque, cash, or RTGS)

·      Fund

·      Department

·      Name of the bank and branch through which the payment is made

·      Account number of the payee

4. Click Search to find the bill, which includes the following information:

·      Name of the payee

·      Payment voucher number

·      Payment voucher date

·      Payment amount

5. Select the check box corresponding to the bill against which a check needs to be assigned.

6. Click Assign Cheque.

 

You get a message indicating that a cheque has been successfully assigned. The page includes the following information:

·      Name of the payee

·      Cheque number

·      Cheque amount

·      Cheque date
