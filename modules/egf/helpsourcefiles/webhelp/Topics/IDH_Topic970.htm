Creating Remittance Recovery

 

The recovery types created in this setup appear as a drop-down list in the "Type" field in the "Deductions" section of the “Contractors Bill Create” screen. On selection of a type, deduction is auto-calculated, and is displayed in the amount column. As many deductions as required can be made from the bill. The net amount after deducting all recoveries from the bill passed amount is automatically credited to the the contractor's account.

 

To record details regarding remittance, perform the following steps:

1. Click Deductions > Remittance Recovery > Create Remittance Recovery to display the Remittance Recovery Create page. This page includes various search criteria to find a remittance.

2. Enter the values as described in the table.

3. Click one of the following options:

·         Save and Close to save the information and close the Window

·         Save and New to save the information and create another entry

·         Cancel to clear the fields and close Window

·         Close to close the Window

 

Table: Remittance Recovery

Field Name

Description

Fund

Name of the fund from which the remittance to be paid.

Recovery Name

Name of the recovery. Example:  CPF, PF, PT

Month

Name of the month against which the remittance is made. For example, to make a PF remittance for an employee for January, select "January" from the drop-down list.

Year

Select an year against which the remittance is made.
