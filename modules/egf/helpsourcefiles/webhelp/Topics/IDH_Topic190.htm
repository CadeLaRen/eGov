Creating/Modifying Functions

 

Function code is taken for each account code being used (debited/ credited) in a transaction. This will be done at the line level in a transaction. Budgets are drawn up and stored at a function level. Function codes can have a hierarchy of any number of levels. Only leaf-level functions (function with no sub-functions) will be used for transactions.

 

You can create a new function and modify the existing ones to suit your requirements.

 

To create/modify a function, perform the following steps:

1. Click Set-up > Functions to display the Function Search page. This page lists out the functions that have been already added.

2. Click one of the following options:

·      To add a new function, click New.

·      To modify a function, click the radio button against the function that you want to Modify.

3. Enter the details as described in the Table below:

4. Click one of the following:

·      Save & New to save the changes and create another function.

·      Save & Close to save the changes and close the window.

·      Cancel to clear the fields and close the window.

·      Close to close the window.

 

 

 

Table: Create/Modify Functions

Field

Description

Function Code

Type a numerical code to uniquely identify the function.

Function Name

Name of the function that you want to create.

Parent Type

Select a parent fund from the drop-down list.

NOTE: You need to select a parent type  only if you want to create a child function under it.

Parent Code

The Parent code automatically appears once the parent Type is selected.

Active

By selecting this check box, the Function becomes active, and accounting of transactions under the function is enabled. By not selecting this,  the Function becomes inactive, and no transactions can be accounted under it.
