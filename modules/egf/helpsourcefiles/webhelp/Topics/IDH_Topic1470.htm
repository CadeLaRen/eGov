Set Up Cheque-in-Hand/Cash-in-Hand

 

This is used to map cheque-in-hand and cash-in-hand accounts within a ULB to administrative units.

For cash and cheque receipts, the right General Ledger code needs to be debited. The specific General Ledger codes can be the same at the city level or more fine-grained at the level of the zones or wards. This is a required setup before receipt transactions can be passed in the system.

 

To set this up, perform the following steps:

1. Click Masters > Set Up Check In Hand/Cash in Hand.

2. Select Create Setup option to display the following fields:

·             Admin level: In most ULBs, this would be the city. In some cases, it may be the Zone.

·             Admin Boundary Value (eg city name, Zone name)

·             Assign the Cash-in-hand and Cheque-in-hand account codes.

 

NOTE: The drop-down account codes for Cash-in-hand and Cheque-in-hand accounts are based on the respective Purpose Codes assigned to General Ledger codes as part of the Account Code setup.
