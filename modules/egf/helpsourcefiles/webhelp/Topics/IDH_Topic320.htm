Viewing Bills

 

You can view the bills that you have previously created.

 

To view a bill, perform the following steps:

 

1. Click Transactions> Expenditure > Bill Registers> View Bill Registers to display the Bill Registers Search page. This page displays various search criteria to view a bill.

2. Enter the details as described in the table and click Search to list the bills that meet the given criteria.

3. To view the details of a bill, click the bill number.

 

Table: View Bill Registers

Field Name

Field Description

Expenditure Type

Select an expenditure from the drop-down list.

Bill Type

Select a bill type from the drop-down list.

Bill Date From

Type the starting bill date

 To

Type the ending bill date

Department

Select a department
