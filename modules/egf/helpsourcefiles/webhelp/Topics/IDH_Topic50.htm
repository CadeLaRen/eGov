Creating Budget Definitions

 

To create a budget definition, perform the following steps:

1. Expand the Budgeting folder to list its subfolders.

2. Expand the Budget Definition folder to show the following hyperlinks:

·      Create Budget Definition

·      Modify Budget Definition

·      View Budget Definition

3. Click Create Budget Definition to show the Budget Definition Create page. This page enables you to enter relevant information regarding the budget that you want to define.

4. Type the name of the budget in the Budget Name field. For example, ULB_Budget_2009-2010.

5. Click one of the following options:

·      BE if you want to create a budget for the next financial year.

·      RE if you want to create a revised budget for the current financial year.

6. Select a financial year for which the budget is defined from the Financial Year drop-down list.

NOTE: The financial year must either be the current year or a future year.

7. Select an option from the Parent drop-down list if the budget that you define belongs to any parent budget.

 

For example, you want to define a budget, ULB_Budget_2009-2010, which is the primary budget of the ULB. This budget will be a combination of individual budgets of each department of the ULB such as Department1Budget, Department2Budget, and so on. When you define a child budget, you must select the name of the parent budget from the Parent drop-down list.

 

8. Type a description about the budget in the Description field.

9. Select the Is Active Budget check box to make the budget active.

NOTE: Transactions can be made only on active budgets.

10. Select the Is Primary Budget check box if the budget that you define is the primary one.

NOTE: Only one budget can be designated as a primary budget for a financial year. A child budget cannot be a primary budget. Also, the parent of a budget estimate (BE) must be a BE.

11. Click one of the following options:

·      Save to save the information.

·      Cancel to clear the fields that you have entered.

·      Close to close the dialog box without saving the details.
