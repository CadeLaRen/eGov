Viewing Budget Groups

 

The Budget Code Search group enables you to view a budget code that has been created earlier.

1. To find a budget code, enter one or more of the following information in the associated fields:

·      Budget code

·      Account type

·      Type of the budget

·      Account code

2. Click Search to list all budget codes that meet the given search criteria.

3. Click the name of the budget you want to view. You get the Budget group View page. This page displays all information regarding the budget group that you want to view.

4. To close the dialog box, click Close.
