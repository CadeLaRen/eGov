Creating/Modifying Funds

 

Fund is a defining concept in municipal accounting – where it is required to segregate all accounting transactions into designated funds and generate a Balance Sheet for each Fund. 

 

Each fund is treated as an independent accounting entity – in other words, all vouchers within a fund must be self-balancing, and balance sheets and Income/Expenditure reports can be generated for each fund. A hierarchy of funds can be defined – i.e. each fund can have multiple sub-funds and so on. Each fund has an identifier – each voucher belonging to a fund has the identifier embedded in the voucher number for easy identification. Fund is taken at a voucher head level for each voucher transaction. This is a mandatory field in all transactions. Only the ‘leaf-level’ nodes are taken in a transaction. These are funds which do NOT have any sub-funds.

 

ULBs are associated with many funds such as General Municipal Fund, Primary Education Fund, and Salary Reserve Fund.

 

To create/modify a fund, perform the following steps:

1. Click Set-up > Funds to display the Fund Search page. This page lists out the funds that have been already added.

2. Do one of the following steps:

·      To create a new parent fund or child fund, click New.

·      To modify a fund, click the radio button against the corresponding parent fund and click Modify. 

 

3. For field description, see the Table below:

 

3. Click one of the following:

·      Save & New to save the changes and create another fund.

·      Save & Close to save the changes and close the window.

·      Cancel to clear the fields and close the window.

·      Close to close the window.

 

Table: Create/Modify a Fund

 

Field

Description

Fund Code

Type a numeric code to uniquely identify the fund.

Fund Name

Name of the fund.

Parent Type

Select a parent fund from the drop-down list.

NOTE: You need to select a parent fund only if you want to create a child fund under it.

Parent Code

The parent code automatically appears once the parent Type is selected.

Identifier

The identifier appears as prefix in all the vouchers accounted in the books of the Fund.

Interfund Transfer Code

Click the Search button and select an inter-fund transfer code.

Active

By selecting this check box, the Fund becomes active, and accounting of transactions under the fund is enabled. By not selecting the check box,  the Fund becomes inactive, and no transactions can be accounted under the Fund.
