Budget Variance Report

 

Budgetary variance Report shows the variance between the budget and actual expenditure as of a given date.

 

To generate the report, perform the following steps:

 

1. In the Reports menu, click Budget Reports > Budget Variance Report.

2. Enter the following details:

·         Department

·         Function

·         Fund

·         Account Type

·         Budget Head

·         As on Date

 

3. Enter the details as described in the table and click Submit to display the report.

 

                                        Table: Budget Variance Report

Field

Description

Department

Department associated with the budget

Function

Function associated with the budget

Fund

Fund associated with the budget

Account Type

Account type associated with the budget

Budget Head

Name of the budget head

As On Date

Date as on when you need to generate the report
