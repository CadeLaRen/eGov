Canceling a Bill Voucher

 

For example, you have created an expense bill voucher which has then been forwarded to the approving officer. The officer can either approve or reject the voucher after verifying the information furnished therein. If the approving officer finds the voucher to be no more required, the officer can reject it leaving a comment saying so. Such vouchers will come to your Inbox for cancellation.

 

To cancel a voucher, perform the following steps:

 

1. Click Transactions> Expenditure >  Cancel Voucher to display the Cancel Bills Search page, displaying the following search criteria to find a voucher:

·      Bill Type

·      Voucher Number

·      Fund

·      Voucher Date From

·      Voucher Date To

·      Department

 

2. Enter the details as described in the table and click Search to list the vouchers according to the given search criteria.

3. Select the check box against the voucher that you want to cancel.

4. Click Submit.

 

You get a message indicating that the voucher has been cancelled.

 

Table: Cancel Bill Voucher

Field

Description

Voucher Number From

Type the lower range of the voucher number

Voucher Number To

Type the upper range of the voucher number

Fund

Fund associated with the transaction

Voucher Date From

Start date of the voucher. For example, to view a bill voucher that was created after Jan 1, 2010, then type 01/01/2010.

Voucher Date To

End date of the voucher. 

Department

Enter the name of the department which created the bill
