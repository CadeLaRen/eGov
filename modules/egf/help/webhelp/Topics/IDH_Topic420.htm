Creating a Scheme

 

You need to create a scheme in the system whenever ULBs undertake any project per a new scheme.

 

To create a scheme, perform the following steps:

1. Click Masters to display the sub-menu tree.

2. Click Schemes > Create Scheme to display the Scheme Create page.

3. Enter the details of the scheme as described in Table 1.

4. Enter the details of the sub-scheme as described in Table 2.

5. Click one of the following options:

·      Save and Close to save the data and close the window.

·      Save and New to save the data and create a new scheme.
