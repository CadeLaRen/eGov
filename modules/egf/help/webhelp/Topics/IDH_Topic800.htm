Viewing Party Types

 

You can view a party type you have previously created.

 

To view a party type, perform the following steps:

 

1. Click Set-up > Party Types > View Party Types to display the Party Type View page.

2. Enter the details as described in the table and click Search to list out the part types you have created.

 

Field Name

Description

Code

Enter a code to uniquely identify a party type. For example, all contractors can have the code-`100', suppliers can have the code -'200'.

Parent Code

Select the parent code if you want to create a child code. For example, 100 (parent) is the code for contractors, and 10010 (child ) is the code for the contractors in the Engineering Department.

Description

Type a brief description about the party type you have created.
