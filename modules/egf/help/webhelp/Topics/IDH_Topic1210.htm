Income-Expenditures Statement Report

 

The income & Expenditure account for a period gives the net surplus or deficit of a ULB during a period.

 

To generate an Income-Expenditures Statement Report, perform the following steps:

1. Click Reports > Financial Statements > Income-Expenditures Statement.

2. Enter the details as described in Table 1.

3. Click Submit to generate the report.

You can export the report in both Excel and PDF formats.

 

Table 1: Income Expenditure Statement Report

Field

Description

From Date*

The starting date.

To Date*

The ending date.

Rupees*

Select an option from the drop-down list to express the amount:

·             Rupees

·             Thousands

·             Lakhs

Department

Select the name of the department from the drop-down list.  

Functionary

Select a functionary.

Function

Select a function.

Field

Select a ward.
