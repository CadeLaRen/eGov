Creating Budget Groups

 

To create a budget group, perform the following steps:

1. Expand the Budgeting folder to list its subfolders.

2. Expand the Budget Group folder to list its subfolders.

3. Click Create Budget Group to show the Budget Group Create page.

4. Type a name for the budget group in the Name field. For example, Department1_SalaryExpense_2009-10.

5. Select the account type that is associated with the budget from the Account Type drop-down list. For example, if the budget group that you create is for salary expenses, you can select Revenue Expenditure from the list.

6. Select one of the following options from the Budgeting Type drop-down list:

·      ALL: This option is used when you need budgeting for all transactions (both debit and credit).

·      CREDIT:This option is usually when you need budgeting only for credit aspect of the transactions.

·      DEBIT: This option is usually used when you need budgeting only for the debit aspect of transactions.

7. Select an account code from one of the following drop-down lists depending on the nature of the transaction:

·      Major Code

·      Min Code

·      Max Code

 

NOTE: For all transactions, you must select an account code. Based on the account code you have selected, the system shows the list of chart of accounts related to the transaction.

 

8. Type a description about the budget in the Description field.

9. Select the Is Budget Group Active check box to make the budget active.

NOTE: You cannot perform any transaction on budget groups that are not active.

7. Click one of the following options:

·      Save to save the information.

·      Cancel to clear the fields.

·      Close to close the window.
