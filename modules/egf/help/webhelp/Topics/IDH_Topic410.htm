Modifying a Supplier/Contractor

 

You can use this feature to modify the details of the supplier/contractor you have stores in the system.

 

To modify a supplier or a contractor, perform the following steps:

1. Click Masters to display the sub-menu tree.

2. Click Supplier/Contractors > Modify Supplier/Contractor to display the Supplier/Contractor Modify page.

3. Enter the code number- of the supplier/contractor you want to modify- in the Code field and and click Go to display the details.

 

You can modify the values to suit your requirements. For field descriptions, see the table below:
