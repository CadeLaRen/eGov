Creating a Supplier/Contractor

 

Whenever ULBs operate with a new supplier/contractor, you have to store their details in the system.

 

To create a supplier or a contractor, perform the following steps:

1. Click Masters to display the sub-menu tree.

2. Click Supplier/Contractors > Create Supplier/Contractor to display the Supplier/Contractor Create page.

3. Enter the values as described in the table.

4. Click one of the following options:

·      Save and New to save the changes and create another supplier/contractor.

·      Save and Close to save the changes and close the window.
