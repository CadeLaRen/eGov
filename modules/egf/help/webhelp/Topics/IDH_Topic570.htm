Creating a Journal Voucher

 

Journal Vouchers are created to record miscellaneous expenses that are not backed by any salary bill, purchase bill, works bill, or contingent bill. JVs can be passed to make adjustment entries as well.

 

You can also use a JV to create vouchers for salary, works, and contractor bills in situations where you have not implemented those modules in the ERP Suite.

 

To create a journal voucher, perform the following steps:

1. Click Transactions > Journal Vouchers > Create Journal Voucher.

You get the Journal Voucher Create page.

2. Enter the details as described in Table 1.

3. In the Journal Voucher Header area, enter the information as described in Table 1.

4. In the Account Details area, enter the information as described in Table 2.

5. In the Sub-Ledger Details area, enter the information as described in Table 3.

6. Click one of the following:

·         Save and New to save the information and create another JV.

·         Save and Close to save the information and close the Window.

·         Cancel to cancel the entries.

·         Close to close the Window

 

Now, the JV will be in the "Unconfirmed" status, and it will be available for approval by the concerned officer. The JVs that have been approved will be in the "Confirmed" status.

 

NOTE: On saving the JV, a budget check will be performed, and a budget appropriation number will be displayed. This will happen only if you create the JV using the "budget check required" codes.

 

Table 1: Journal Voucher

Field Name

Description

Voucher Number

The number is automatically displayed

Voucher Date

The current date is displayed. You can change the date to suit your requirement.

Voucher Sub-Type

Select an option from the drop-down list to specify the type of voucher that you want to create. (Eg., salary bill voucher, works bill voucher)

Party Bill Number

Bill number of the party (for instance, supplier bill number)

Party Name

Name of the party

Party Bill Date

Date of creation of the bill by the party

Fund

Name of the fund through which a payment is made

Department

Name of the department that made the payment

Scheme

Name of the scheme associated with the payment

Subscheme

Name of the subscheme

Functionary

Name of the functionary

Fund Source

Name of the fund

Field

Ward number

Narration

Brief description about the JV

 

Table 2: Account Details

Field Name

Description

Function Name

Name of the function

Account Code

Account code associated with the transaction

Account Head

This field is automatically displayed

Debit Amount

The amount to be debited

Credit Amount

The amount to be credited

 

Table 3: Sub-Ledger Details

Field Name

Description

Account Code

Type the sub-ledger account code. This will be the same account code that you have entered in the Account Details.

Type

Select the type of the voucher from the drop-down list. For example, contractor.

Code

Code number of the party. If it's a contractor voucher, then type the code number of the contractor.

Name

Name of the party automatically appears in this field.

Amount

The bill amount
