Modifying Salary Codes

 

Occasionally, you may want to modify the salary codes that you have previously created.

 

To modify a salary code that you have created, perform the following steps:

1. Click Set-up > Salary Codes > Modify Salary Codes to display the Salary Codes Modify page, where all salary codes that you have created are listed.

 

This page lists all the salary codes that you have created. You can modify them to suit your requirement.

2. Click Back to go back to the previous page.

 

Table: Modify Salary Code

Field Name

Description

Head

Name of the pay head. Example: Basic Pay

Account Code

Select the account code from the chart of accounts

Account Description

This field is automatically displayed as it appears in chart of accounts

Type

Select one of the following options:

·             Earning - if the pay head is an earning component such as HRA (House Rent Allowance)

·             Deduction- if the pay head is a deduction component such as PF (Provident Fund)
