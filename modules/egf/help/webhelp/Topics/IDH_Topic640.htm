Recording Bank-to-Bank Transfer Details

 

Bank-to-bank transfer is a special service that allows you to transfer funds electronically to accounts in other banks. When you make a transfer, you need to record the details in the system to track the transaction.

 

To record a bank-to-bank transfer details, perform the following steps:

1. Click Transactions > Contra Entries > Cash Deposit to display the Cash Deposit page.

2. Enter the details as described in the table.

3. Click Save to save the information.

 

Table 1: Header Details

Field Name

Description

Voucher Number

Type the voucher number

Voucher Date

Type the voucher date

Check in Hand

This field is displayed automatically

Cash in Hand

This field is displayed automatically

Bank

Name of the bank where you deposited the cash.

Voucher Date

The current date is displayed. You can change the date.

Account Number

Your account number with the bank where you deposited the cash

Amount

The amount that you have deposited in the bank

Fund

The name of the fund is automatically displayed, depending on the bank you have selected.

Department

Select the name of the department associated with the transaction.

Scheme

Select the name of the scheme associated with the cash deposit.

Subscheme

Select a subscheme if any.

Functionary

Select a functionary from the drop-down list.

Fund Source

Select a fund source from the drop-down list.

Field

Select a ward number associated with the fund. Example: Property Tax for Ward 10.
