Creating an Opening Balance

 

Opening balances need to be set up during the first year of operation when historical data is migrated into the system.

On an ongoing basis, opening balances for any year will be automatically calculated by the system. However,  in case any new opening balances need to be set (e.g. when an old asset that was earlier not in the books is valuated by the ULB), then this feature can be used.

 

NOTE: For an account code, only a credit OR a debit opening balance may be setup.  The list (i.e. sub-ledger codes) is from the values assigned under the accounting entity mapped to the account code.

 

To create an opening balance, perform the following steps:

1. Click Period-End Activities > Opening Balance to display the Opening Balance Setup page.

2. Enter the details as described in the Table.

3. Click one of the following:

·         Submit to save the opening balance.

·         Cancel to cancel the entry.

·         Close to close the screen without saving your work.

 

 

                              Table: Create Opening Balance

 

Field Name

Description

Fund Name

Select a fund from the drop-down list.

Financial Year

Select a financial year from the drop-down list.

Type

Select Asset/Liability from the drop-down list.

Major Head

Select Major Head from the drop-down list.

Minor Head

Select Major Head from the filtered drop-down list under the major code selected above.

Financing Source

Select from the drip-down list of financing Source.

Account Code

Select from the filtered list of account code under the selected major and minor codes.

Account Head

On selection of the Account Code, the Account Head automatically appears in this field.

Accounting Entity

For an account code without an accounting entity, the words “No Entity” automatically appears in this field. For an account code with accounting entities, the drop-down list of accounting entities appears, from which the relevant accounting entity has to be selected.

Debit Amount/Credit amount

The opening balance has to be entered in the respective field, depending on whether the balance is a debit balance or credit balance.
