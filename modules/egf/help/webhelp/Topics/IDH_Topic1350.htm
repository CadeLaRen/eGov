Generating Cheque Issue Register

 

This report includes details on the cheques that have been issued to a supplier or contractor.

 

To generate the Cheque Issue Register, perform the following steps:

1.   Click Reports > MIS-Cheque Issue Register

2.   Specify the details as mentioned in the following table:

 

Field

Description

Bank name

Enter the name of the bank from which the cheque is issued.

Account Number

Enter the bank account number

Cheque From Date

Click the Calendar icon to select a date or type in the date from when you want to generate the report.


Cheque To Date

Click the Calendar icon to select a date or type in the date till when you want to generate the report.

 

3.   Click Submit to generate the report.

    

NOTE: You can export the report in both Excel and PDF formats.
