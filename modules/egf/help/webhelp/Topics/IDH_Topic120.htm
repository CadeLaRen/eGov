Adding Budget Details

 

To add budget details, perform the following steps:

1. Expand the Budget Details folder to show its subfolders.

2. Click Create Budget Details to show the Add Budget Details page.

3. Select the name of the budget- whose details you want to enter- from the Budget drop-down list. The name of the budget is what you gave when you defined the budget.

4. Select a department which executes the project from the Executing Department drop-down list.

 

For example, assume that you have defined a budget for the Health department to build a municipal hospital, which will be executed by the Engineering department. In this case, you must select Engineering from the drop-down list.

 

5. Enter a date in the As on Date field. The default date will be the current one.

 

When you click Get Actuals, the system automatically pulls out the actual figures of the selected budget group for the previous two financial years and display them in the Actuals fields. For example, assume that Miscellaneous Expense is the budget group for which you want to add the budget details. Assume that the actual miscellaneous expenses for the previous financial years (eg., 2007-08 and 2008-99) were 15000 and 22000 respectively. When you click Get Actuals, these values will be displayed in the Actuals (2007-08) and Actuals (2008-09) fields.

 

6. Select a budget group that was created for the selected budget from the Budget Group drop-down list.

 

For example, for the Department1Budget, you have created two budget groups; Department1_Expenditure and Department1_Revenue. Now, you need to enter the budgeted amounts for each of these budget groups.

 

7. Perform the following steps:

 

a.   Select a function from the Function drop-down list.

b.   Select a fund from the Fund drop-down list (eg., Capital Project Fund).

c.    To get the actuals as of a specific date for the current financial year as well as for the previous two financial years, type the date in the As On Date field and click Get Actuals. The actual amount appears in the corresponding fields.

d.   In the Anticipatory Upto 31 March field, type the amount that is expected to be spent or earned pertaining to the selected budget group by March 31 of the current financial year. For example, if you have selected "Department1_Expenditure" as a budget group, then you need to specify the expenditure that is anticipated to be incurred by Department1 for the current financial year.

e.   In the Estimate Amount field, type the budgeted amount for the selected budget group for the next financial year.

f.     To add details of one more budget group of the same budget, click the Plus icon and perform the steps (a) through (e).

g.   To add details of the budget group of a different budget, click Save and New.

8. Click one of the following options:

·      Save and Continue to add budget details of the same budget group

·      Save and New to add budget details of a different budget group

 

NOTE: If you click Save and Close, the budget will be available for further modifications to the person who created it and is not sent for checking. You can find this budget available for modifications on the Modify Budget Details page.
