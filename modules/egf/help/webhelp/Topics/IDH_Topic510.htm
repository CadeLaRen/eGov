User-Defined Codes

 

Once an entity is defined, user-defined codes can be set up; under that, the system will then need to maintain sub-ledger details for each such user-defined code.

 

To create a code for the accounting entities that you have created, perform the following steps:

 

1. Click Masters to display the sub-menu tree.

2. Click Scheme > User-Defined Codes to display the User-Defined Code create page.

3. Select an entity from the SubCode For drop-down list.

4. Click New to display the User-Defined Codes Create/Modify page where you can type the relevant information.

5. Enter the details as described in the table.

6. Click one of the following options:

·      Save and New to save the code and add another code.

·      Save and Close to save code and close the window.

 

NOTE: The added code will be listed in the chart of accounts under the respective general ledger.
