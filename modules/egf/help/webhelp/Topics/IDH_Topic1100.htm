Creating a Report Schedule

 

You can configure the schedule for the following statutory reports:

·      Balance Sheet

·      Income/Expenditure Statement

·      Receipts/Payment Statement

 

To configure the schedule, perform the following steps:

1. Click Set-up > Report Schedule Mapping > Create Report Schedule to display the Schedule Definition page.

2. Enter the details as described in the table.

3. Click one of the following options:

·      Save and New to save the changes and schedule another report

·      Save and Close to save the changes and close the window

 

Table: Report Schedule
