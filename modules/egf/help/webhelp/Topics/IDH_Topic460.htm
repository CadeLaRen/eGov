Creating/Modifying Account Codes

 

You can create account codes in the system whenever the situation warrants.

 

To create/modify a detailed account code, perform the following steps:

 

1. Click Masters to display the sub-menu tree.

2. Click Chart of Accounts > Create/Modify/View Detailed Code to display a page displaying the following options:

·      Search and Modify

·      Add New

·      Close

3. Do one of the following steps:

·      To add a new account code, click Add New.

·      To modify the details of an account code such as its type and name, type the account code in the Account Code field and click Search and Modify. You can modify all details except the account code.

·      To close the window, click Close.

4. Enter the details as described in the table.

5. Click one of the following:

·      Save to save the details.

·      Close to close the Window

 

The account code that you have created will be listed under the corresponding levels.

 

Table: Create/Modify Account Codes

 

Field Name

Description

Parent GL Code

GL code of the parent entry. For example, you want to create a detailed code under Income->Tax->Property Tax. Then, you need to type the GL code of the Property Tax.

Account Code

Thew new account code automatically appears in this field.

Name

Name of the account for which you create the code

Description

Brief description on the account.

Purpose

The reason for creating the new detailed code.

Function Required

Select this check box if function is required for the new account.

Budget Required

Select this check box if all transactions associated with this account require a budgetary check. Selecting this check box will prevent you from creating the transaction, if the budget is not available.

Account Detail Type

Select an option to indicate the beneficiary of the created account code

Active for Posting

Select this check box if transaction is ready to be posted.
