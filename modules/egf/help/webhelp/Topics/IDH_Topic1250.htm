Cash Book Report

 

Cash book report gives a chronological account of all cash receipts and payments, during a selected period, with opening and closing cash balances.

 

To generate a Cash Book Report, perform the following steps:

1. Click Reports > Financial Statements > Cash Book.

2. Enter the details as described in Table 1.

3. Click Submit to generate the report.

You can export the report in both Excel and PDF formats.

 

Table 1: Cash Book Report

Field

Description

Boundary *

This field is defaulted to the name of the ULB.

Fund

Name of the associated fund.

Starting Date*

Starting date.

End Date*

Ending date.

Department

Select the name of the department from the Department drop-down list.

Functionary

Name of the functionary.

Function

Name of the function.

Field

Ward number.

Show Condensed Report

Select this check box to generate a condensed report.
