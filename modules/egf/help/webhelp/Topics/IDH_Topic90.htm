Modifying Budget Groups

 

To modify a budget group, perform the following steps:

1. Expand the Budgeting folder to list its subfolders.

2. Expand the Budget Group folder to list its subfolders.

3. Click Modify Budget Group to show the Budget Group Search page.

4. Type the name of the budget group- you want to modify- in the Name field. For example, Department1_SalaryExpense_2009-10.

5. Click Search to display the budget group.

6. Click the name of the budget in the Name column to display the details you want to modify.

7. You can change the value of each field. For field descriptions, see Creating Budget Groups.

8. Click one of the following options:

·      Save to save the change.

·      Cancel to clear the fields.

·      Close to close the window.
