Generating a Bank Payment Voucher (BPV)

 

A BPV must be generated against all confirmed purchase/contractor/C-bill/salary bill vouchers.

 

To generate a BPV, perform the following steps:

1. Log on to the application using the appropriate login credentials.

2. Click Transactions > Expenditure > Payments > Bill Payment.

 

You get the Bill Payment Search page which includes the following search criteria to find a bill:

·      Bill Number

·      Bill Date From

·      Bill Date To

·      Expenditure Type

·      Fund

·      Fund Source

·      Department

 

3. Enter values in the relevant fields and click Search to display a page which has the following four tabs:

·      Expense Bill

·      Contractor Bill

·      Supplier Bill

·      Salary Bill

 

4. Navigate to the relevant tab to list out the confirmed purchase/contractor/C-bill/salary bill vouchers.

 

The bill vouchers include the following information:

·      Bill Number

·      Bill Date

·      Voucher Number

·      Voucher Date

·      Name of the Payee

·      Net Amount

·      Payment that was made earlier

·      Payable Amount

·      Fund associated with the transaction

·      Name of the department which made the transaction

·      Fund Source

 

5. Select the check box against the voucher whose payment needs to be made.

 

6. Select one of the following options to specify a payment mode:

·      Cheque

·      Cash

·      RTGS

 

7. Click Generate Payment to display the Bill Payment page where you need to provide the following information:

·      Voucher number

·      Voucher date

·      Bank and branch details

·      Bank account number

 

8. In the Approval Information section, enter the information as described in Table 2. This section will be available only if you choose to use a manual workflow.

 

NOTE: Using the eGov Admin module, you can set up either a manual or an automatic workflow that suits best for your business requirements. If you have chosen to use a manual workflow, while creating the bill, you need to select the department and designation of the person who needs to approve it. When you use an automatic workflow, you do not need to manually select these fields. The system automatically forwards the bill voucher to the relevant users for approval.

 

9. Click Save and Forward to send the payment voucher to the Inbox of the specified officer for approval. A message appears on your screen indicating the designation and user name of the officer to whom the payment voucher has been forwarded.
