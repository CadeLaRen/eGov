CPF Setup

 

Setting up the Employer PF Contribution process allows the user to map the tax deduction source code and expense code. Based on the CPF set up you can trigger the employer contribution to provident fund.

 

To set up the contribution provident fund, perform the following steps:

1. In the Payroll menu, click Provident Fund.

2. Click CPF Setup.

3. Enter the details as described in Table 12.

 

Table 12: CPF Setup

Field

Description

*Tds

Select the tax deduction source from the Tds drop-down list.

*CPF Expense Account

Type the contribution Provident Fund account details in the CPF Expense Account field.

 

The field next to CPF Expense Account field displays payhead details of the expense account.

*Rule Script

Select the rule script from the Rule Script drop-down list.

 

4. Click one of the following options:

·      Submit: To set up the contribution provident fund

·      Cancel: //////////////////////Option button not working////////////////////

·      Close: To close the screen without saving the details
