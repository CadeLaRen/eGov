Setting up a Provident Fund Account

 

In order to set up Provident Fund (PF) details for a ULB, a PF Liability Account and a PF Interest Expense Account are required. Both these accounts are created and managed in the eGov Financials module. The Chart of Accounts for the ULB contains the account names and account numbers.

 

To set up a PF Account, perform the following steps:

1. In the Payroll menu, click Provident Fund.

2. Click PF Setup.

The Provident Fund screen is displayed.

3. Type the PF liability account number in the PF Liability Account field.

 

NOTE: This account number is created in eGovernment Financials and needs to be obtained from the Chart of Accounts. After the account number is typed, the name of this account is displayed in the adjacent text box.

 

4. Type the PF Interest Account Number in the PF Interest Expense Account field.

 

NOTE: This account number is created in eGovernment Financials and needs to be obtained from the Chart of Accounts. This is the account where the PF interest is credited. After the account number is typed, the name of this account is displayed in the adjacent text box.

 

5. Click Save to open the PF Interest Rates screen.

6. Select the frequency from the Frequency drop-down list.

7. Enter the details as described in Table 11.

8. To specify another period with a different interest rate, click Add Row.

9. To remove any row, place the cursor on that row and click Delete Row.

10. To save the details you have entered, click Save.

11. To close the screen, click Close.

 

Table 11: PF Interest Rates

Field

Description

Date From*

Type the date from which the PF rate of Interest comes into effect.

You can enter the date also by clicking the calendar icon.

Date To

Type the date till which the PF rate of Interest comes into effect.

Annual Rate of Interest*

Type the annual rate of interest.
