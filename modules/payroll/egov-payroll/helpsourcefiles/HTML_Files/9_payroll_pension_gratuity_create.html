Create Gratuity Details

The Gratuity Processing screen calculates the total payment that needs to be paid to an employee or a nominee. This includes the pension and the grauity amount.

In the Gratuity Processing screen:

1. The following employee details are displayed:

  • Employee Code
  • Employee Name
  • Designation
  • Department
  • Birth Date
  • Superannuation Date

2. The following gratuity disbursement details that are displayed can be edited:

  • Disbursement Type
  • Bank
  • Bank Branch
  • Bank Account

NOTE: The values displayed are from the employee record of eGov EIS-Payroll.

2. In the Pension Sanction Number field, type the pension sanction number.

3. In the Sanction Authority field, type the name of the employee authorised to sanction the payment.

4. The Pension Commuted (%) field displays the value of the commute cap. You can type the percentage of the employee basic that is required for the commuted pension. However, this cannot exceed the commute cap. If no pension needs to be commuted, type 0.

The following details are displayed based on the ULB:

  • Commuted Period (years)
  • Commuted Amount
  • Gratuity Factor
  • Gratuity Amount
  • Total Payment (Gross)

5. In the Monthly Pension Payable field, the calculated monthly amount is displayed. This amount can be edited.

6. If the employee is deceased before the retirement date, the following nominee details are displayed:

  • Name of the nominee
  • The total payment that needs to be disbursed to the nominee(s). If the nominees are the employee's children, the total amount is divided equally among them.
  • Monthly Pension Payable

7. Click Save to save the details. This creates a pension number and a bill number for the employee.
