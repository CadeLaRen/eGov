Create Recovery Details

1. In the Record Recoveries screen, the following information is displayed from the employee record:

  • Employee Code
  • Employee Name
  • Designation
  • Department
  • Employee Fund

2. If there are any outstanding payments that need to be made from to other accounts, the following message is displayed:

Recoveries are there in another fund, need to run IUTs. This requires an Inter-Unit Transfer (IUT) to be passed manually. See About Recovery Payments for details.

The following is displayed in the Record Recovery screen when the Journal Vouchers are passed:

  • Account Head is the account number for which an outstanding payment exists
  • Amount of the outstanding payment
  • Reference number of the recovery document

3. To enter a recovery manually, click the + icon and add the details.

4. To delete a recovery, click the - icon.

5. Click Save to save the details.
