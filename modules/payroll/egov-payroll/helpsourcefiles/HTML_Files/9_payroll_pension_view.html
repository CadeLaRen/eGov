View Pension Details

You can view the following pension details of the employee:

  • Employee Code
  • Name
  • Designation
  • Department
  • Date of Birth
  • Superannuation Date
  • Pensioner Status
  • Disbursement Type
  • Details of all the nominees such as full name, their relationship with the employee, whether alive, employed, married, and so on.

See Create Pension for field descriptions.
