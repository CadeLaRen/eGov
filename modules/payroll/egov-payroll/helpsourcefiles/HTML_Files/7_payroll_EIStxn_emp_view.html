View Employee Details

To view employee details:

1. In the EIS menu, select Employee.

2. Click View to open the Search Employee to View screen. Search for the employee record based on any of the criteria given below:

  • Designation: Click the designation of the employee from the drop-down list.
  • Department: Click the department of the employee from the drop-down list.
  • Employee Code: Type the employee code, if known.

    NOTE: When a search is based on the Employee Code, the search is done on all active and inactive employee records. In such a case, the Search All option is invalid.

  • Employee Name: Type the employee name, if known.
  • style="margin-left:0;"Search All (Active and Inactive): Click the radio button beside the appropriate option:
    • Yes to search for all employees (both past and present). It is not advisable to select this option as it could result in a large number of recrods being retrieved which could make the application very slow.
    • No to search for only active employees.

3. Click Submit. Employees that satisfy the criteria are displayed. See Create Employee for field descriptions.

4. Click Close after you view the details.
