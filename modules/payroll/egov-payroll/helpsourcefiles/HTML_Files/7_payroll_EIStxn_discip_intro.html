Disciplinary Action

If an employee conducts himself in an inappropriate manner, a grievance can be filed against the employee. An enquiry is launched that is headed by an Enquiry Officer who decides the punishment, if applicable.

eGov EIS Payroll tracks and maintains the charges against an employee and the disciplinary action taken. You can create, view, and modify a disciplinary action. To do this:

1. In the EIS menu, select Disciplinary.

2. Click Create, View, or Modify as appropriate.

3. Search for the employee using one of the following search criteria:

  • Designation
  • Department
  • Employee Code
  • Employee Name
  • Search All

4. Click Submit. A list of employees matching the search criteria is displayed.

5. Click the View or Modify link as applicable.
