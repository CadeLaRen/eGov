Log in to eGov Applications

Each user is provided a unique UserName and Password.

1. In the UserName field, type your UserName.

2. In the Password field, type your password.

3. Click Login.

The system then validates these details. When a valid UserName and Password combination is entered, the application home page is displayed.

The home page is divided into the following parts:

  • The Header Section which shows the following:
    • Name of the ULB
    • eGovernments logo. If a user clicks on this when in any other screen, they return to the home page.
    • Logout link. Click this when you have finished your work and wish to exit from the system.
  • Menu Tree Section: This lists the menu in a tree form. The actual contents of the tree are populated based on the role assigned to a particular user. eGov applications are fully menu-driven with a multi-level menu structure which allows a user to select an option in the menu and go to the next lower level with each selection.
