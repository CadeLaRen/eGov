Create Details for Gratuity Disbursal

In order to disburse the gratuity to a former ULB employee, a set of details associated to the disbursal needs to be recorded. These details are specified in the Disburse Gratuity screen.

1. The following details are displayed in the Disburse Gratuity screen:

  • Employee Code
  • Employee Name
  • Birth Date
  • Superannuation Date
  • Designation
  • Department
  • Fund
  • Disburse Gratuity to
  • Payment Method

    NOTE: If the employee is deceased, the only valid payment method type is cheque.

2. In the Voucher No field, type the voucher number for the approved gratuity.

3. In the Voucher Date field, type the date the voucher was created in the dd/mm/yyyy format.

NOTE: You can also click the calendar icon beside the Voucher date field to select the date on which the voucher was created.

4. In the Payer Bank drop-down list, click the bank name from which the ULB issues the gratuity.

5. In the Payer Bank Branch drop-down list, click the branch name from which the ULB issues the gratuity.

6. In the Bank Account drop-down list, click the bank account of the payer. This is the account from which the gratuity is issued.

7. In the Payee details section, the following are displayed:

  • Name of the payee. This can be one of the following:
    • If the employee receives the gratuity the name of the employee is displayed.
    • If the employee is deceased, the name of the employee nominee(s) who are eligible is displayed.
    • If the disbursement type is cash, Commissioner is displayed because only a Commissioner is authorised to disburse gratuity in cash.
  • Relationship with the employee.
  • Cheque Amount. This displays the gratuity amount that needs to be disbursed.
  • Payee Bank. For payments that are made by direct bank transfers, the employee's bank to which the gratuity is transferred is displayed.
  • Payee Bank Branch. For payments that are made by direct bank transfers, the employee's branch is displayed.
  • Payee Bank Account. For payments that are made by direct bank transfers, the employee's bank account is displayed.

NOTE: If there are multiple eligible nominees, the gratuity amount is divided equally among them.

You need to enter the following details:

  • Cheque No: Click the arrow icon above the Cheque No field. This displays the next available cheque that can be used.
  • Cheque Date: Type the date the cheque is created.

NOTE: You can also click the calendar icon beside the Cheque Date field to select the cheque date.

8. Click Save to save the details for gratuity disbursement. The following message is displayed: Executed Successfully.
