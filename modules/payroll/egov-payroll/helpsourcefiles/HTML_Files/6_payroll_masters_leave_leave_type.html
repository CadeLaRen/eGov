Leave Master: Create Leave Type

1. In the EIS menu, select Leave Masters.

2. Select Leave Type Create. This opens the Leave Type Create screen.

3. You need to provide the details mentioned in the table.

Field Name Field Value Description
Leave Type Name* Character The name of the leave type must be entered. If you type the name of a leave type that has already been created, a message stating this is displayed.
Is Half Day Radio button You can select one of the following:

Click Yes to indicate that it is possible for an employee to avail of half-day leave for this leave type.

Click No to indicate that it is not possible for an employee to avail of half-day leave for this leave type.
Pay Eligible Radio button You can select one of the following:

Click Yes to indicate that the employee is entitled to be paid when availing this leave type.

Click No to indicate that the employee is not entitled for pay when availing this leave type.
Accumulative Radio button You can select one of the following:

Click Yes to indicate that it is possible for an employee to accumulate this leave type.

Click No to indicate that it is not possible for an employee to accumulate this leave type.

4. Click Save to save the leave type details for the new leave type created.
