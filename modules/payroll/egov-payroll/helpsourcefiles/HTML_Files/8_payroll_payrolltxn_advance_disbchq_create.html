Create Cheque Disbursement Details

In the Create Disbursement by Cheque screen, a list of all the sanctioned advances for the employee with the disbursement type as cheque is displayed.

1. From the list of advances, select the Pay check box adjacent to the advance that needs to be disbursed. More than one advance disbursal can be selected. The TOTAL field displays the total amount that needs to be disbursed for the employee.

2. In the Voucher No. field, type the voucher number.

3. Type the Voucher Date in the dd/mm/yyyy format. You can also click on the calendar icon to select the date.

4. In the Paid By drop-down list, click the relevant cashier name.

5. From the Bank drop-down list, select the bank from which the disbursement needs to be made. The banks in the list belong to the same fund as the employee.

6. From the Bank Account drop-down list, select the associated bank account. The Balance amount is displayed.

7. Click the arrow mark to display the next available cheque number from the cheque book that is used currently.

8. Type the Cheque/DD date in the dd/mm/yyyy format. You can also click on the calendar icon to select the date.

9. Click one of the following:

  • Save & Close: To save the advance disbursement details and close the screen.
  • Save & New: To save the advance disbursement details and create a new disbursement record.
  • Save & View: To save the advance disbursement details and view the disbursement details.
  • Cancel: To clear all the entries in the screen.
  • Show GL Entry: To display the General Ledger entries for the advance.
  • Close: To close the screen without saving details.
