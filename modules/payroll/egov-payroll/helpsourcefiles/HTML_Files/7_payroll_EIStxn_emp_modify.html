Modify Employee Details

To modify an employee record in EIS:

1. In the EIS menu, select Employee.

2. Click Modify to open the Search Employee to Modify screen.

3. Search for the employee account based on any of the criteria given below:

  • Designation: Click the employee’s designation from the drop-down list.
  • Department: Click the employee’s department from the drop-down list.
  • Employee Code: Type the employee code. When you search on a particular employee code, the Search All option is ignored.
  • Employee Name: Type the employee name.
  • Search All (Active and Inactive): Click the radio button beside the appropriate option:
    • Yes to search for all employees (both past and present). It is not advisable to select this option as it could result in a large number of recrods being retrieved which could make the application very slow.
    • No to search for only active employees.

4. Click Submit to search for the employee record. Employees that satisfy the search criteria are displayed.

5. Click Modify in the row that has the employee name whose account you wish to modify. The Employee Details screen of the selected employee is displayed.

6. Modify the details of the employee as required. However, the employee's user name cannot be changed.

To assign a new payscale or another designation to the employee:

  • Click Add Row in the Payscale Details section and enter the new payscale details. See Create Employee for field descriptions. Only the latest payscale is used during payroll calculation for the employee.
  • Click Add Row in the Assignment Details section and enter the new designation details. See Create Employee for field descriptions.

NOTE: Click Delete Row to delete a designation and all its details. You need to have a minimum of one entry pertaining to assignment details.

7. In the Modification Summary field, briefly mention the changes you made to the employee account.

8. You can now:

  • Click Save to save all the modifications.
  • Click Close to close the Employee Details screen without saving any details.
