Create Employee

Employee details are entered in the Create Employee screen to create an employee record.

1. Enter the personal details of the employee in the Employee Personal Details section.

NOTE: All field names that are superscripted with an asterisk are mandatory, values must be provided in them.

See the following table for field descriptions:

Employee Personal Details
Field Description
Status To select the status of the employee, do one of the following:
  • Select Employed when the employee is currently employed at the ULB (in service). A payslip needs to be generated for such employees.
  • Select Deceased when the employee is deceased.
  • Select Retired when the employee has retired from the services of the ULB. Such an employee is eligible to receive a pension.
  • Select Suspended when the employee is suspended. Such employees will not have a payslip generated.
Employee Code Automatically generated for a new employee.
First Name* Type the first name of the employee.
Middle Name Type the middle name of the employee.
Last Name Type the last name of the employee.
Father/Husband's First Name Type the first name of the employee's father or husband.
Middle Name Type the middle name of the employee's father or husband.
Last Name Type the last name of the employee's father or husband.
Date Of Birth* Type the employee's date of birth in the (dd/mm/yyyy) format. For example, if the date of birth is 1st February 1980 type 01/02/1980.
Gender Click the button beside the gender of the employee.
Blood Group To select the blood group of the employee, click the relevant option in the drop-down list.
Mother Tongue Type the mother tongue of the employee.
Religion To select the religion of the employee, click the relevant option in the drop-down list.
Community To select the community of the employee, click the relevant option in the drop-down list.
Is Physically Handicapped? Click Yes if the employee is physically handicapped.

Click No if the employee is not physically handicapped.
Is Medical Report Available? Click Yes if the medical report of the employee is available.

Click No if the medical report of the employee is not available.
Identification Mark-1 If the employee has an identification mark (such as a birthmark or scar), type a description of this.
Identification Mark-2 If the employee has another identification mark, type a description of this.
PAN Number Type the PAN number of the employee.
Permanent Address Type the permanent address of the employee.
Correspondence Address Type the employee's address for correspondence.
Languages Known To select the language known to the employee, click the relevant option in the drop-down list. To select multiple languages, press the SHIFT key and click on all the languages known to the employee.
Local Language Qualified Click the local language in which the employee's qualification degree was attained.

2. Enter the user name details in the User Details section.

User Details
Field Description
Is User Active Click Yes to activate the user name of the employee.
User Name* Enter a user name for the employee. This is one of the requirements for the employee to log in to the eGov application. However, the employee also needs to be given a user role in the Admin Module.

To log in with the above user name, the password needs to be in the format, employee first name underscore employee code. So if the employee's first name is Usha and her employee code is 1190, her password would be: Usha_1190

The password can be changed in the Admin Module.
Remarks Enter any comments

3. After you have filled in the necessary details, click one of the following options:

  • Save to save the personal details of the employee.
  • Close to clear all the details and close the screen.

4. To enter assignment details of an employee, see Entering Assignment Details.

Entering Assignment Details

1. Go to the Assignment Details tab.

2. Enter the assignment details as described in the following table:

Assignment Details
From Date Enter the date when a specific position is assigned to the employee
To Date Enter the last date when the employee held the position
Fund Enter the fund with which the employee was associated.

Example: Capital Fund Project
Function
Grade Select the grade of the employee from the drop-down list.
Designation Enter the designation that is held by the employee during the period
Position Enter the position that is held by the employee during the period
Functionary
Main Department Enter the main department to which the employee belongs
Govt Order No Enter the order number that is given by the government
If Head of Department Specify whether the employee is the head of the department
Remarks Enter any other remarks that you want to make

3. After you have filled in the necessary details, click one of the following options:

  • Save to save the assignment details of the employee.
  • Close to clear all the details and close the screen.

NOTE: After you click Save, the assignment details that you have entered appear in the Assignment Detail List section.

4. Do one of the following steps:

  • To add another assignment, click the Plus icon.
  • To delete an assignment, click the Cross icon.

5. To enter service history of the employee, see Entering Service History.

Entering Service History

1. Go to the Service History tab.

2. In the Service History/Ledger Details section, do the following steps:

  • To add any comments about the service of the employee, type the comments in the Comments field.
  • To specify the date of comment, type the date in the Comment Date field.
  • To specify the reason for the comment that is made, type the reason in the Reasons field.

3. Do one of the following steps:

  • To add another comment, click the Plus icon.
  • To delete one comment, click the Cross icon.

4. In the Probation section, do the following steps:

  • To specify the post the employee holds during probation, select an option from the In Which Post drop-down list.
  • To specify the date when probation was declared, type the date in the Probation Declared field.
  • To specify the order numer via the employee is put on probation, type the order number in in the Order Number field.
  • To specify the order date, type the date in the Order Date field.

5. Do one of the following steps:

  • To add another row, click the Plus icon.
  • To delete one row, click the Cross icon.

6. In the Regularisation section, do the following steps:

  • To specify the post the employee holds during regularization, select an option from the In Which Post drop-down list.
  • To specify the date when probation was declared, type the date in the Probation Declared field.
  • To specify the order numer via the employee is reguralized, type the order number in in the Order Number field.
  • To specify the order date, type the date in the Order Date field.

7. Do one of the following steps:

  • To add another row, click the Plus icon.
  • To delete one row, click the Cross icon.

8. To specify any remarks, type the remarks in the Remarks field.

9. Do one of the following steps:

  • To save the information, click Save.
  • To close the window without saving the information, click Close.

10. To enter other employee information such as educational qualification and property details, see Entering Other Personal Information.

Entering Other Personal Information

1. Go to the Other Details tab.

2. To enter educational qualifications of employees, in the Educational Qualification section, do the following steps:

  • Specify the degree obtained by the employee in the Qualification field. Example: Bachelor of Technology.
  • Specify the specialized subjected in the Major Subject field. Example: Computer Science.
  • Specify the date when the degree was obtained in the Pass Date field.
  • Specify the university or board from where the degree was obtained in the University/Board field.

3. Do one of the following steps:

  • To add more degree, click the Plus icon.
  • To delete one degree, click the Cross icon.

4. To enter technical qualifications of employees, in the Technical Qualification section, do the following steps:

  • Specify a skill from the Skills drop-down list.
  • Specify the grade obtained in the Grade field.
  • Specify the date when the degree was obtained in the Pass Date field.
  • Specify additional comments in the Remarks field.

5. Do one of the following steps:

  • To add more degree, click the Plus icon.
  • To delete one degree, click the Cross icon.

6. To enter details about departmental tests entertaken by employees, in the Departmental Test section, do the following steps:

  • Specify the name of the test from the Name of the Test field.
  • Specify the date when the employee cleared the test in the Pass Date field.

7. Do one of the following steps:

  • To add details about another departmental test, click the Plus icon.
  • To delete test details, click the Cross icon.

8. To enter details about the immovable property that is possed by employees, in the Immovable Property Details section, do the following steps:

  • Specify the description about the property in the Property Description field.
  • Specify the place of the property in the Place field.
  • Select an option from the How Acquired drop-down list to indicate how the property was acquired by employees.
  • Specify the present value of the property in the Present Value field.
  • Specify the details of permission that is obtained from the government in the Permission Obtained field.
  • Specify the order number of the property in the Order Number field.
  • Specify the date when the order was obtained in the Order Date field.

9. Do one of the following steps:

  • To add details about another immovable property, click the Plus icon.
  • To delete one property, click the Cross icon.

10. To enter details about the movable property that is possed by employees, in the Movable Property Details section, do the following steps:

  • Specify the description about the property in the Property Description field.
  • Specify the value of the property at the time of purchase in the Value of Time of Purchase field.
  • Select an option from the How Acquired drop-down list to indicate how the property was acquired by employees.
  • Specify the details of permission that is obtained from the government in the Permission Obtained field.
  • Specify the order number of the property in the Order Number field.
  • Specify the date when the order was obtained in the Order Date field.

9. Do one of the following steps:

  • To add details about another movable property, click the Plus icon.
  • To delete one property, click the Cross icon.

10. To enter any additional remarks, type comments in the Remarks field.

11. Do one of the following steps:

  • To save the information, click Save.
  • To close the window without saving the information, click Close.
