Modifying Payscale Details

To modify a Payscale, perform the following steps:

1. In the Payroll menu, click Payroll Masters.

2. Click Payscale, and then click Modify.

Search PayScale screen is displayed.

3. Select the payscale you want to modify from the PayScale Name drop-down list.

4. Click Modify PayScale to modify the following payscale details:

  • Payscale name
  • Pay Commission
  • Effective From
  • Amount From/To
  • Payhead name
  • Type
  • Calculation Type
  • % Basis
  • Amount
  • Increment From
  • Increment To
  • Increment

5. Make necessary modifications and mention the changes that are made in the Remarks field.

6. Click one of the following options:

  • Save & View to save the changes and view the new payscale details.
  • Cancel: To cancel the changes that were made

NOTE: If any changes are made while calculating a payslip that is being processed manually, you need to click Recompute Deductions to display the modified values.
