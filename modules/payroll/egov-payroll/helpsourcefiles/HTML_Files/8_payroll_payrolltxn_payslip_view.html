View Employee's Payslips

1. The View Payslips screen displays all the payslip related information such as:

  • Employee Code
  • Employee Name
  • Department Name
  • Designation
  • Payscale Name
  • Attendance
  • Earnings such as Basic Pay, DA, HRA, and so on
  • Deductions such as Provident Fund, Taxes, and so on
  • Gross Pay
  • Net Pay

2. You can select one of the following:

  • Click PRINT. This prints the payslip displayed.
  • Click Close. This closes the payslip displayed.
