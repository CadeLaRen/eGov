Search Screen

1. Search for the employee record using any of the following search criteria:

Field Name Field Value Description
Designation Drop-down list Click the designation of the employee.
Department Drop-down list Click the department of the employee.
Employee Code Numeric Type the Employee Code. When you start to enter the Employee Code, a list of all employees with employee codes that begin with the number you typed is displayed. Click the employee code from the list. The employee name is displayed automatically.
Employee Name Character Type the name of the employee.
Status Drop-down list Select one of the following:
  • Click Employed when the employee is working in the ULB.
  • Click Deceased when the employee is deceased.
  • Click Retired when the employee is retired.
  • Click Suspended when the employee is suspended.

2. After the search criteria are entered, click Submit. The list of records that match the search criteria that are entered is displayed.

NOTE: If your Search results contain more than one hundred records, an error is displayed. You will need to refine your search.

3. To create, view, or modify details, click the Create, View, or Modify link beside the relevant employee name.
