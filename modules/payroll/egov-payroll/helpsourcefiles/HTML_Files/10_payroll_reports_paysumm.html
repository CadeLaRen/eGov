Generating a Pay Summary Report

The Pay Summary report shows employee pay details for a given financial year. When this report is generated for the current financial year, the details till date are displayed.

To generate the Pay Summary report of an employee, perform the following steps:

1. In the Payroll menu, select Reports.

2. Click Pay Summary.

Pay Summary Report screen is displayed.

3. Type the employee code in the Employee Code field.

4. Click the financial year-for which the report needs to be generated-from the Financial Year drop-down list.

5. Click Submit to generate the Pay Summary report.

The following details are displayed in the Pay Summary

  • Employee Code
  • Employee Name
  • Employee Designation
  • Date of Joining
  • Earnings and Allowances--breakup of individual earnings and allowances and their total
  • Deductions--breakup of individual deductions including loan recoveries and their total
  • Net Pay
