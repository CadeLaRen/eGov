Managing Payheads Details

Payheads define the salary components of an employee payslip. A payhead can be defined using a unique set of attributes that need to be defined only once. A payhead once defined can be mapped to many payscales.

These payscales are subsequently assigned to an employee and are used during payslip calculations.

Payheads are defined in Payhead Masters.
