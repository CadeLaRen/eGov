Fill Details in Leave Application Form

1. Enter details in the Leave Application form as described in the table.

Field Description
Leave Type To select the type of leave the employee has applied for, click the relevant option from the drop-down list.

The leave types in the drop-down list are those applicable for the employee designation.

Reason Type the reason the employee has applied for leave.
From Date Type the starting date from which the employee wants to apply for leave in the dd/mm/yyyy format.

For example, to apply for leave from 1st March 2008, type 01/03/2008.

NOTE: If another leave application for the employee contains this date, an alert is displayed.
To Date Type the end date for which the employee wants to apply for leave in the dd/mm/yyyy format.

For example, to apply for leave until 15th March 2008, type 15/03/2008.

NOTE: If another leave application for the employee contains this date, an alert is displayed.
Working Days Displays the number of working days the employee has applied for leave.
Available Leaves Displays the number of leave days available for that leave type.
Is Encashment? Displays a check mark if the leave type is Encashment.

NOTE: This option is displayed only for certain leave types.

Taking 2 half leaves in a day Click Yes if the leave applicant wants to apply for two half day leaves on the same day. Click No if the leave applicant does not want to apply for two half day leaves on the same day.

NOTE: This option is displayed only for leave types where it is specified that the leave can be taken in half-day periods.

2. After you enter the details, click Approve.

A confirmation message, Executed saving Leave Application successfully, is displayed.

3. Click OK.

Important Note: If leave is applied for a period in the past for which the employee's pay processing has been done, the payslip will not reflect this. If there are any pay adjustments that arise due to this, these have to be made through arrear payslips.
