Viewing Bulk Updation Details

To view the bulk payhead details that are created, perform the following steps:

1. In the Payroll menu, click Bulk Updation.

2. Click View.

Search Bulk Updation Rule page is displayed.

3. Select the month- for which you want to see the bulk updation details- from the Month drop-down list.

4. Select the financial year- for which you want to see the bulk updation details- from the Financial Year drop-down list.

5. Click Submit.

Depending on the search criteria, a list of payheads that are bulk updated is displayed.

6. To view a payhead click the view link in the relevant row.

Bulk updation details are displayed.

7. Click Close to close the page.
