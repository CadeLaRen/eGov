Viewing a Payhead

To view a Payhead, perform the following steps:

1. In the Payroll menu, click Payroll Masters.

2. Click Payhead and then click View.

Search Payhead screen is displayed.

3. Select the name of the payhead you want to view from the Payhead Name drop-down list.

2. Click View to view the following payhead details:

  • Name
  • Order
  • Description
  • Type
  • Category
  • Calculation Type
  • % Basis
  • Recovery Account Name
  • Recovery Account Code
  • Account Code
  • Account Name
  • Is attendance based
  • Is recomputed
  • Is recurring
  • Interest payable
  • Taxable payable

3. Click Print to print the payhead details.
