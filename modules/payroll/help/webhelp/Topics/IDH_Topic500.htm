CPF Trigger

 

To trigger the CPF for a batch of employees, perform the following steps:

1. In the Payroll menu, click Provident Fund.

2. Click CPF Setup.

3. Enter the details as described in Table 13.

 

Table 13: CPF Trigger

Field

Description

*Tds

Tax deduction source is displayed in the Tds field.

*CPF Expense Account

Contribution Provident Fund account details is displayed in the CPF Expense Account field.

 

The field next to CPF Expense Account field displays payhead details of the expense account.

*Rule Script

The rule script is displayed in the Rule Script drop-down list.

*Month

Select the month for which you want to set up the CPF in bulk from the Month drop-down list.

*Financial Year

Select the financial year for which you want to set up the CPF in bulk from the Financial Year drop-down list.

 

4. Click one of the following options:

·      Trigger: To trigger the CPF for the batch of employees

·      Close: To close the screen without saving the details
