Approving a Payslip

 

After employee payslips are generated (manually or by batch generation), the payslips have to be approved by the concerned officer. Only a ULB employee with the required privileges can approve a payslip. The concerned officer can access the payslips pending for approval from the Inbox.

 

In the Payslip Approval screen, payslips that are pending approval are grouped based on the employee department and the month in which the payslip was generated. Each grouped set of payslips is displayed with a task. A task indicates the number of payslips that are pending approval for each set of payslips.

 

To approve a payslip, perform the following steps:

1.   Log on to the Payroll application. You get the Inbox where all tasks that need your approval are listed with the following information:

·    Date and time at which you received the task in your Inbox

·    Position and user name of the sender

·    Name of the task

·    Status of the task

·    Other details
 

2.   Click the task (Payslip) you want to approve. You get the approval page displaying the task details which you can verify for its correctness.

NOTE: Before approval or rejection you must enter a comment in the Comment field.

The payslip details can be modified as required and saved. See, Modifying an employee payslip.
 

3.     Click one of the following options:

·    Approve: To approve the payslip. 

·    Reject: To reject the payslip.
 

4.   Click Submit. You get a confirmation message.
