Pay Bill Summary Report

 

The Pay Bill Summary report shows the number of days the employee is absent and the amount deducted based on the number of days of absence.

 

To generate the Pay Bill Summary report of an employee, perform the following steps:

1.   In the Payroll menu, select Reports.

2.   Click Pay Bill Summary.

3.  Select the month for which the report needs to be generated from the Month drop-down list.

4.  Select the year for which the report needs to be generated from the Year drop-down list.

5.  Select the department for which the report needs to be generated from the Department drop-down list.

6.  Select the functionary for which the report needs to be generated from the Functionary drop-down list.

7.  Click Submit to generate the Pay Bill Summary report.
