Creating a Nomination Head Master

 

To create a Nomination Head Master, perform the following steps:

 

1.   In the Payroll menu, click Nomination Head Master.

2.   Click Create. The Create Nomination Header page is displayed.

3.   In the Code field, enter the employee code.

4.   In the Description field, enter the payhead description.

5.   From the PayHead drop-down list, select a payhead.

6.   In the GL Code field, enter the General Ledger code number.

7.   In the GL Name field, enter the name of the General Ledger.

8.   From the Rule Script drop-down list, select a rule script.

9.   Click Save.

         A Nomination Head Master is created successfully.
