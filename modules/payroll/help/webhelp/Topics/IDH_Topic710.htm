Updating Employee Status in EIS

 

Since employees can receive pension only if they are retired or deceased, the employee status has to be updated in the Employee Information System (EIS).

 

To update employee status in the EIS, perform the following steps:

1.   Click EIS from the left navigation bar.

2.   Click Employee.

3.   Click Modify to display the Search Employee page.

4.   Specify any of the following employee details:

·      Designation

·      Department

·      Employee Code

·      Employee Name

·      Employee Status

 

5.   Click Submit. A list of employees and their details are displayed in a grid according to the search criteria.

6.   Click Modify from the Masters column. The Employee Details page is displayed.

7.   In the Employee Details page, do the following steps:

a)   From the Employee Status drop-down list, click one of the following options:

·      Retired: if the employee is retired.

·      Deceased: if the employee is deceased

b)   In the Permanent Address field, enter the permanent address of the employee.

c)   In the Remarks field, enter a brief description about the modifications that you have made.

d)    Click Save to save the details.

 

After the employee status is updated, you have to check if the pension details exist for the employee. Pension details include nominee details and disbursement method for the employee. For more information, see Creating Employee Pension Header Data.
