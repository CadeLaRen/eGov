Modifying Gratuity before approval

 

937:Gratuity can be modifed while approving.

 

The transaction screen from inbox -

 

    * Will have modify, approve, reject option.

    * On approve, the object is sent to next level .

    * On Reject, the the gratuity calculation gets cancelled. (As of now, the process may vary based on requirement)

    * On modify the gratuity calculation screen is shown in modify mode for user to update the calculation. The script based calculation will not be triggered at this time. User can only do the manual changes allowed.
