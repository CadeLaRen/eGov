Creating Employee Pension Header Data

 

To create the employee pension data, perform the following steps:

1.   From the Payroll menu, click Pension.

2.   From the Employee Master list, click Create. The Payroll Search Screen page is displayed.

3.   Specify any of the details required to find an employee as described in the following table:

 

Table 1: Pension Search

Field

Description

Designation

Select the designation of the employee from the from the Designation drop-down list.

Department

Select the department to which the employee belongs from the Department drop-down list.

Employee Code

Enter the employee code.

Employee Name

Enter the name of the employee.

Status

Select the employment status from the Status drop-down list.

Functionary

Select the functionary from the Functionary drop-down list.

Type

Select the type of employment from the Type  drop-down list.

 

4.   Click Submit. A list of employees and their details are displayed in a grid according to the search criteria.

5.   Click Create from the Masters column. The Employee Pension Details page is displayed.

6.   The following employee details are automatically populated:

·      Code

·      Name

·      Designation

·      Department

·      Date of Birth

·      Super Annuation Date

·      Deceased Date

·      Pensioner Status

7.   From the Disbursement Type drop-down list, select any one of the following options:

·      Cash: if the type of disbursement is in cash.

·      Bank Transfer: if the type of disbursement is through a bank transfer.

NOTE:  If you have selected Bank Transfer, enter the following bank details:

·      Bank: Select the name of the bank.

·      Branch Name: Select the name of the branch where the bank is located.

·      Account Number: Enter the bank account number of the employee.

·      Cheque: if the type of disbursement is in cheque.

8.   Select any one of the following options:

·      Yes: to assign a nominee.

·      No : select and click Save.

The employee pension header data is successfully created.

 

Assigning a Nominee

 

After selecting the Yes option, the Nominee Details section is displayed.

In the Nominee Details section, do the following steps:

1.   From the Nominee Name drop-down list, select the name of the nominee.

NOTE: The Nominee Name drop-down list has values or names that have been initially created in the Employee Information System (EIS).
 

2.   The following field are automatically populated based on the data in the EIS.

·      Relationship

·      Alive

·      Married

·      Date of Birth

·      Employed

·      Disbursement Type

·      Bank

·      Branch Name

·      Account Number

3.   In the Percentage field, enter the percentage of the pension amount to be allotted to the respective nominee.

4.   Click Update Status.

The Status field gets populated as either eligible or not eligible based on the system rules. These rules include the following:

·      wife is eligible and gets precedence

·      daughter is eligible until she is married

·      son is eligible only if he is unemployed


NOTE: Not E indicates Not Eligible. If the selected nominee is not eligible then the system automatically inserts a zero in the Percentage field.

 

5.   Click Save.

The nominee details are saved successfully.


NOTE: Click the Plus (+) icon to add another nominee. Similarly, click the Cross (X) icon to delete a nominee.
