Creating a Payhead Rule

 

To create a payhead rule, perform the following steps:

1. In the Payroll menu, click Payroll Masters.

2. Click Payhead Rule and then click Create.

Create Payhead Rule page is displayed.

3. Select the payhead- for which you want to define a rule- from the Salarycode drop-down list.

4. Specify the date from which the payhead is on effect- in the Effective from field.

5. Specify a description about the payhead rule in the Description field.

6. Specify the path of the file where the rule is defined in the Rule File Path field.

7. Click one of the following options:

·  Submit to save the payhead rule details entered

·  Cancel to cancel the details entered
