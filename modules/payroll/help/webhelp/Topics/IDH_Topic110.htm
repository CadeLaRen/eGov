Creating Payscale Details

 

To define a payscale, perform the following steps:

1. In the Payroll menu, click Payroll Masters.

2. Click Payscale, and then click Create.

Payscale Create screen is displayed.

3. Enter the details as described in Table 2.

4. Click one of the following options:

·      Save & View: To save the values entered and to view the same

·      Save & New: To save the payscale details and open a new screen for creating another payscale

·      Cancel: To cancel the values entered

 

Table 2: Creating Payscale Masters

*Payscale Name

Type the name of the payscale.

 

NOTE: It is recommended to name the payscale based on the payscale range. For example, consider the maximum amount of the Basic component is Rs. 5000 and the minimum amount of the Basic component is Rs. 3000, then you can use 3000-5000 as the Payscale Name.

 

If the increment for this payscale is Rs. 50, you can also use 3000-50-5000 as the Payscale Name.

Pay Head

Select the name of the relevant payhead component from the Payhead drop-down list.

Calculation Type

Based on the payhead type Calculation Type will be displayed.

% Amount

Based on the payhead type % Amount is displayed

NOTE: The Calculation Type field and the % Amount field cannot be edited. To modify these, see Modify Payheads.

%Basis

Based on the payhead type % Basis is displayed.

 

NOTE: The Calculation Type field and the % Basis field cannot be edited. To modify these, see Modify Payheads.

Amount

Perform the following steps:

·      When the Calculation Type is Monthly Flat Rate, type the amount in the Amount field.

For example, suppose the payhead component Basic for a payscale is Rs. 10,000. When you click the payhead component as Basic, in the Calculation Type field: Monthly Flat Rate is displayed. To specify the Basic as Rs. 10,000, type 10000 in the Amount field.

·      When the Calculation Type is Computed Value, type the percentage value in the % Amount field.

For example, suppose the payhead component DA needs to be calculated as 10% of the Basic. When you click the payhead component as DA, in the Calculation Type field, Computed Value is displayed and in the % Basis field, Basic is displayed.To assign DA as 10% of Basic, type 10 in the % Amount field.

·      When the Calculation Type is Slab-Based, the Amount is taken based on the slabs defined in the Recovery account. This is defined in the relevant payhead.

 

NOTE: To add/delete a payhead row, click the Add/Delete icons.

Increment Details

Amount From

Type the minimum payscale value.

Amount To

Type the maximum payscale value.

Increment Amount

Type the increment amount.

For example, to add an increment of Rs. 50 to the payscale 3000-5000, type 3000 in the Amount From field, type 5000 in the Amount To field and type 50 in the Increment Amount field.

 

To assign multiple increment values, click the + icon. When adding multiple increment values, ensure that:

·      The first value in the Increment From field should be the same value as the minimum payscale.

·      The last value in the Increment To field should be the same value as the maximum payscale.

·      All Increment values lie within the payscale range.

·      Each Increment From value should begin at the number that immediately succeeds the preceding Increment To value.
