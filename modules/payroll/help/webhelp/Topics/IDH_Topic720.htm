Creating a Pension Record in EIS

DELETED

If pension details do not exist or have to be updated in the EIS, perform the following steps:

1.   Click EIS from the left navigation bar.

2.   Click Employee.

3.   Click Modify to display the Search Employee page.

4.   Specify any of the following employee details:

·      Designation

·      Department

·      Employee Code

·      Employee Name

·      Employee Status

 

5.   Click Submit. A list of employees and their details are displayed in a grid according to the search criteria.

6.   Click Modify from the Masters column. The Employee Details page is displayed.

 

1. In the EIS-Payroll menu, click EIS.

2. Click Pension.

 

3. Click Create, View, or Modify to open the Search Employee screen.

 

4. Search for the employee based on any of the following criteria:

 

    * Designation

    * Department

    * Employee Code

    * Employee Name

    * Employee Status

 

5. Click Submit. This lists the employees that satisfy the search criteria.

 

6. Do one of the following:

 

    * Click Create to define pension details for the employee.Select this when no previous record exists.

    * Click View to view pension details of the employee. Select this to view pension details for an employee.

    * Click Modify to modify pension details of the employee. Select this when the pension details need to be updated.}
