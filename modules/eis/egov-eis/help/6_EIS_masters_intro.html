EIS Masters

In order to work with the EIS features it is necessary to have several EIS Masters set up. Some of these masters are:

  • Leave Master
  • Personal Information Masters such as:
    • Blood Group
    • Category
    • Employee Status
    • Property
    • Pay Fixed
    • Religion
    • Local Languages Qualified
    • Community
    • Departmental Test
    • Recruitment Mode
    • Languages Known
