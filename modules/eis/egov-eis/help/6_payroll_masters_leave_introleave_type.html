Leave Types

A leave type is created in a Leave Master. It defines the type of leave that can be availed by an employee. When a leave type is created, you can:

  • Specify the number of leave days designation-wise.
  • Assign specific leave types as paid leaves.
  • Enable specific leave types to be carried over to the next financial year, if not availed.
  • Allow half-day leave for specific leave types.

To create, view, or modify a leave type:

1. In the EIS menu, select Leave Masters.

2. Do one of the following:

  • To create a leave type, select Leave Type Create.
  • To view a leave type, select View Leave.
  • To modify a leave type, select Modify Leave.

You can specify the number of leave days based on the employee's designation. To do this:

1. In the EIS menu, select Leave Masters.

2. Select Designation Leave Mapping.

The number of unavailed leave days at the end of the financial year can be added to the same leave type for the next financial year. A leave type can be eligible for carry-over leave only if it is defined in the Leave Master. See Create Leave Master for details.

To create or update the opening balance of a specific leave type:

1. In the EIS menu, select Leave Masters.

2. Select Create/Update Leave Opening Balance.
