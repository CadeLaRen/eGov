Create Pension Record

The Employee Pension Details screen displays the employee details from the employee record. To create a pension record, you need to enter the pension disbursement method and details of the employee nominee.

To enter the details:

1. In the Disbursement Type drop-down list, click one of the following:

  • Bank Transfer: Select this to transfer the pension amount directly to the employee's bank account.
  • Cheque: Select this when the pension needs to be disbursed by cheque.
  • Cash: Select this when the pension needs to be disbursed by cash.

NOTE: This cannot be edited for an employee who is deceased.

2. If the Disbursement Type is Bank Transfer, the following details are displayed:

  • Bank
  • Branch Name
  • Account Name

You can edit these details.

3. You can create a nominee for the employee. See Create a Nominee for details.

4. Click Save to save the pension details.

Create a Nominee

Nominees receive pension only when the ULB employee is deceased.

If the employee's spouse is alive, he or she becomes the sole nominee for the employee and will receive the pension amount.

If the employee's spouse is deceased, the following criteria need to be fulfilled by the employee's son or daughter to qualify as a nominee:

  • Son: Should be alive, under 25 years of age, and unemployed.
  • Daughter: Should be alive, under 25 years of age, unemployed, and unmarried.

To specify nominee details:

1. In the Employee Pension Details screen, click Yes in the Create a Nominee option.

2. Under Nominee Details, enter the following details for the nominee:

  • First Name: Type the first name of the nominee.
  • Middle Name: Type the middle name of the nominee.
  • Last Name: Type the last name of the nominee.
  • Relationship: Click the relationship of the nominee with the ULB employee.
  • Alive: Select the relevant option from the drop-down list.
  • Married: Select the relevant option from the drop-down list.
  • Date of Birth: Type the date of birth of the nominee in the dd/mm/yyyy format.
  • Employed: Select the relevant option from the drop-down list.
  • Disbursement Type: Select the disbursal method. For the Disbursement Type Bank Transfer, enter the following details for the nominee:
    • Bank
    • Branch Name
    • Account Number

3. Click Add Row to add additional nominees.

4. Click Update Status to verify if the nominees are eligible for pension. The Status field for the employee displays one of the following:

  • Eligible
  • Not Eligible

Only eligible nominees receive pension.

5. Click Save to save the nominee details.
