Leave Master: Create or Update Leave Opening Balance

Search for the Leave Type of an employee to create or update the opening balance:

1. In the Designation drop-down list, click the designation of the employee from the drop-down list.

2. In the Department drop-down list, click the department of the employee from the drop-down list.

3. In the Employee Code field type the employee code. The Employee Name is displayed.

4. In the Leave Type drop-down list, click the name of the leave type.

5. In the Financial Year drop-down list, click the current financial year.

6. Click Submit to display the Employee name with the Leave Opening Balance.

7. Do the following:

  • To create the opening balance, type the number of leave days that are unavailed in the previous year. This is added to the leave days of the current year.
  • To modify the opening balance, edit the number of leave days.

8. Click Save to save the details.
