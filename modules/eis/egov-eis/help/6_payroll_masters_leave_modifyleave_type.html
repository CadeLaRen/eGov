Leave Master: Modify Leave Type

1. In the EIS menu, select Leave Masters.

2. Select Modify Leave.

3. From the drop-down list beside Choose Leave Type Name, select the Leave Type that needs to be modified.

4. Click Submit.

5. You can change all of the following:

  • Leave Type Name: Enter a new Leave Type Name or make changes to the Leave Type Name.
  • Is Half Day: Click Yes to indicate that it is possible for an employee to avail of half-day leave for this leave type.Click No to indicate that it is not possible for an employee to avail of half-day leave for this leave type.
  • Pay Eligible: Click Yes to indicate that the employee is entitled to be paid when availing this leave type. Click No to indicate that the employee is not entitled for pay when availing this leave type.
  • Accumulative: Click Yes to indicate that it is possible for an employee to accumulate this leave type. Click No to indicate that it is not possible for an employee to accumulate this leave type.

6. Click Save to store the new details for this Leave Type.
