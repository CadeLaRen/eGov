EIS Reports

EIS Reports provide details of Employee History, Leave, Attendance, Pay Summary, and Payslips Generated.

To access the Employee History, Employee Attendance, or Leave report:

1. In the EIS menu, click EIS Reports.

2. From the options, click Employee History/Employee Attendance/Leave. This opens the Employee Search screen.

3. Enter details of the employee or set of employees for whom you wish to search.

  • To search for a single employee, type the Employee Code or Employee Name.
  • To search for multiple employees, select either the Designation or Department or both.

4. Click Submit.

  • For Employee History: A screen that shows the employee or multiple employees is displayed. Click the Employee History link to view the employee's details. See Employee History Report.
  • For Attendance: Attendance Report is displayed.
  • For Leave: Leave Report is displayed

Pay Summary Report

To access the Pay Summary report:

1. In the Payroll menu, select Payroll Reports.

2. Click Pay Summary to open the Pay Summary screen.

3. In the Employee Code field, type the employee code.

4. Click the financial year for which the report needs to be generated from the drop-down list.

5. Click Submit to generate the Pay Summary report. See Pay Summary.

Payslips in PDF Format

To access the Payslips in PDF format:

1. In the Payroll menu, select Payslip.

2. Click View Payslips-PDF format. This displays the Generate PDF for Payslips screen.

3. In the Month/Year field, click the month and the financial year for when the payslip should be viewed.

4. To view the PDF department-wise, click the department from the drop-down list.

5. If changes to payslips have been made after a PDF has been generated, and you need to view the latest changes, select the Force Generation of PDF Payslips check box.

NOTE: Doing a Force Generation of PDF Payslips could take a long time.

6. Click Generate PDF to display the payslips in the PDF format. See View Payslips in PDF.
