Attendance Report

This report can be generated for a specific department and a given month. The report can be used to verify employee attendance prior to payslip generation.

The Attendance Report shows:

  • Employee Name
  • Number of days in the month
  • Number of days absent
  • Number of unpaid leave days
  • Number of days present
  • Number of paid leave days
  • Number of CompOffs in the month
  • Number of working days in the month

The details in this report cannot be changed.
