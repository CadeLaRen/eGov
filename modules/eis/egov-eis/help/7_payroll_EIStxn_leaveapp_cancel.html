Cancel Leave Application

A leave application can be cancelled when an employee does not take leave on the dates mentioned in the leave application. For example, when an employee has applied for 5 days leave but returns to work after 3 days.

In this situation, the original leave application made for 5 days needs to be cancelled and a new application for 3 days needs to be created. Partial cancellation of a leave application is not possible.

To cancel a leave application:

1. In the EIS menu, select Leave Application.

2. Click Cancel. This opens the Search Leave Application to Cancel screen.

3. Enter the name, employee code, or other search criteria that will help to locate the employee for whom it is necessary to cancel the leave.

4. From the search results, identify the employee for whom you want to cancel leave that has been applied for.

5. In the employee record, click the cancel link in the Masters column. This displays the leave applications for the employee.

6. In the rightmost column of the row where the leave application is to be cancelled, click on the link to open the Leave Application screen. This shows all the details of the leave to be cancelled.

7. Click Cancel. A pop-up with the message, Do you want to cancel the leave? is displayed.

8. You can now:

  • Click Cancel to retain leave application.

    Or

  • Click OK to cancel the leave. The confirmation message, Leave application cancelled successfully is displayed.

After the leave application is cancelled, all the leave dates that have been cancelled are now blank in the Attendance Register. The dates that are blank should now be marked with the letter P.

After this is done, a fresh leave application for the actual leave dates needs to be made. After this leave application is approved, the Attendance Register will contain L for the relevant dates.

NOTE: If an employee payslip has already been generated taking into account the leave application that was later cancelled, an arrear payslip needs to be generated in the Payroll Module.
