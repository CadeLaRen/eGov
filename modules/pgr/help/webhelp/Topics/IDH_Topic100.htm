Creating a Complaint Type

 

To create a Complaint Type, perform the following steps:

1.   Log on to the PGR System.

2.   Expand the PGR folder to list its subfolder.

3.   Expand the PGR Master folder icon to list its subfolders.

4.   Expand the Complaint Type folder icon to display the Create and Update links.

5.   Click Create to display the Create Complaint Type page.

6.   Specify the following:

·         Complaint Type Name:type the name of a complaint.

·         Complaint Type Name Local: type the name of the complaint in local language.

·         Department Name: select the name of the department to which the complaint sbelongs from the drop-down list.

·         Complaint Group: select the name of the complaint group to which the complaint belongs from the drop-down list.

·         Escalation Duration: type the duration in days within which if a complaint is not resolved then it will get escalated to the next higher official.

7.   Click Add.You get an acknowledgement message:

Complaint Type Master was created.

8.   Click Close to close the window.

 

 

Editing a Complaint Type

 

To edit a Complaint Type, perform the following steps:

1.   Log on to the PGR System.

2.   Expand the PGR folder to list its subfolder.

3.   Expand the PGR Master folder icon to list its subfolders.

4.   Expand the Complaint Type folder icon to display the Create and Update links.

5.   Click Update to display the Edit Complaint Type page.

6.   From the Complaint Type Name drop-down list, select the name of the type of complaint you want to edit.

7.   Click Edit to display the Edit Complaint Type page.

8.   Specify the following:

·         Complaint Type Name: type the new name of a complaint.

·          Complaint Type Name Local: type the new name of type of complaint in local  language.

·          Department Name: select the name of the department to which the complaint belongs from the drop-down list.

·          Complaint Group: select the name of the complaint group to which the complaint belongs from the drop-down list.

·          Escalation Duration: type the new duration in days within which if a complaint is not resolved then it will get escalated to the next higher official.

9.   Click Update.You get an acknowledgement message:

Complaint Type Master was updated.

10. Click Close to close the window.
