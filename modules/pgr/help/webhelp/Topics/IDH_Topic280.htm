Finding Complaints Assigned to You

 

This feature is used by a city official who is either a Grievance Officer or a Municipal Officer to find the list of complaints assigned.

 

To find the list of complaints, perform the following steps:

1.   Log on to the PGR System.

2.   Click the List My Complaints hyperlink. You get a page that displays the number of complaints and the complaint details in tabular form.

3.   Click any of the following column titles in the table to sort data in the table:

·    Complaint Number

·    Complaint Type

·    Complaint Title

·    Department

·    Location

·    Status

·    Registration Date

·    Expiry Date

4.   Click a specific complaint number to view the details of that complaint.

 

NOTE: Complaint details in the following sections cannot be edited:

·    Complaining Person's Details

·    Complaint Details

·    Complaint Messages

 

5.    The Other Options section can be edited as follows:

·    To change the complaint status, select an option from the Change Status drop-down list. The complaint status list includes: Registered, Forwarded, Processing, Completed and Rejected.

·    To change the jurisdiction zone, select the appropriate zone from the Zone drop-down list

·    To change the jurisdiction ward, select the appropriate ward from the Ward drop-down list.

·    To change the department, select the appropriate department from the Change Dept drop-down   list.

·    To change the complaint type, select the appropriate complaint type from the Change  Complaint  Type drop-down list.

·    To forward the complaint to a specific authority, select the appropriate official from the Forward   Complaint To drop-down list.

·    To add more information regarding a complaint, enter a message in the Post Message text box.

6.   Click Submit. The page is now saved and the complaint with new details added is displayed. The complaint is forwarded to the official in the appropriate department.
