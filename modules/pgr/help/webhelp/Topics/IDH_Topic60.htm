Changing Password

 

This feature is used by city officials to change their passwords, whenever they are willing to do so.

 

To change the password, perform the following steps:

1.    Log in to the PGR system.

2.    Click the Change Password hyperlink. You get the Change Password page with the following fields:

·    Username -this cannot be edited

·    Old Password

·    New Password

·    Confirm New Password

3.   Enter the old password in the Old Password field.

4.   Enter the new password in the New Password field.

5.   Enter the new password again in the Confirm New Password field.

6.   Click Change Password. The password is then changed.
