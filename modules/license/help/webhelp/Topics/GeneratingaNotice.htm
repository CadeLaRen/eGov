Viewing Notices

Previous Top Next 

Notice is a template that includes some specific information regarding a property.
Using PTIS, you can generate the following notices:

 

·      Notice 6: This is generated when you register a new property.

·      Notice 7: This is generated when there is an increase in property tax.

·      Notice 9: This is generated when there is a transfer of ownership of a property.

·      Notice10: This is generated when there is a decrease in property tax.

You must provide the citizen with the printout of the relevant notice whenever there is a change in the property details. You can generate and print a notice at any point in time.

 

To print/view a notice, perform the following steps:

1.   Click Property Tax from the left navigation bar..

2.   Click PTIS Reports.

3.   Click Search Notices to display the Search Notice page.

4.    From the Notice Type drop-down list, select one of the following options:

·     Notice6

·     Notice7

·     Notice9

·     Notice10

5.   Type one of the following:

·     Notice Number        

·     Bill No           

·     From Date and To Date

 

NOTE: For viewing the notice by date, both the the fields From Date and To Date are mandatory.

   

6.   Click Search to display a list of properties.

7.    Click the notice number against the property whose notice you want to view/print.
