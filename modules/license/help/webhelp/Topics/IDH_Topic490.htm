Create New Hospital License

Previous Top Next 

To create a new Hospital license perform the following steps:

1.      Log on to the Hospital License as creator.

2.      Click Hospital License on left navigation bar.

3.      Click Transactions to display New Hospital license.

4.      Click Create New License to display New License set up screen.

5.      Specify the License details as described in the following table:
