Create Water works License

Previous Top Next 

To create a new trade license perform the following steps:

1.      Log on to Water works license as a creator.

2.      Click Water works License on left navigation bar.

3.      Click Transactions to display New water works license.

4.      Click Create New license to display New License set up screen.

5.      Specify the License details as described in the following table:
