Generating a List of Properties Marked for Deactivation

Previous Top Next 

To generate a list of properties marked for deactivation, perform the following steps:

1.   Click Property Tax from the left navigation bar.

2.   Click PTIS Reports.

3.   Click Properties Marked For Deactivation to display the List Of Properties Marked For  Deactivation page.

4.   From the Zone drop-down list, select the zone number to which the property belongs.

5.   From the Ward drop-down list, select the ward number to which the property belongs.

6.   Click List to display a list of properties marked for deactivation in the selected zone and ward.

7.   Click a particular bill number of a property to view the details.
