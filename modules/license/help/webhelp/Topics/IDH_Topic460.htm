Create Electrical Contractor License

Previous Top Next 

To create a new Electrical Contractor license perform the following steps:

1.      Log on to Electrical Contractor license as a creator.

2.      Click Electrical Contractor License on left navigation bar.

3.      Click Transactions to display New Electrical Contractor license.

4.      Click Create New license to display New License set up screen.

5.      Specify the License details as described in the following table:
