Finding a Property

Previous Top Next 

You can find a registered property using any one of the following:

·     Bill Number

·     Mobile Number

·     Zone and Ward

·     Area and Location 


To find a property, perform the following steps:

1.   Click Property Tax from the left navigation bar.

2.   Click Existing Property.

3.   Click Search Property to display the Search Property page.

4.   Do one of the following steps:

·    To find a property using the Bill Number, do the following steps:

a. In the Zone field, type the zone number to which the property belongs.

b. In the Ward field, type the ward number to which the property belongs to.

c. In the Bill No. field, type the four-digit bill number of the property.

d. In the Sub No. field, type the sub number in case the property is divided earlier.

e. Click Search to display the Search Results page.


·    To find a property using the Mobile Number, do the following steps:

a. In the Mobile No. field, type the ten digit mobile number of the property owner.

b. Click Search to get the Search Results Page.


·    To find a property using the Zone and Ward, do the following steps:

a. From the Zone drop-down list, select the number of the zone to which the property  belongs.

b. From the Ward drop-down list, select the ward number to which the property belongs.

c. From the Street Name drop-down list, select the name of the street to which the property belongs.

d. In the Owner Name field, type the name of the property owner.

e. In the Door No. New field, type the new door number of the property.

f.  In the Door No. Old field, type the old door number of the property.

g. Click Search to get the Search Results page.


·    To find a property by using the Area and Location, do the following steps:

a. From the Area drop-down list, select the name of the area to which the property belongs.

b. From the Locality drop-down list, select the name of the locality to which the property belongs.

c. From the Street Name drop-down list, select the name of the street to which the property belongs.

d. In the Owner Name field, type the name of the owner of the property.

e. In the Door No. New field, type the new door number of the property.

f.  In the Door No. Old field, type the old door number of the property.

g. Click Search to display the Search Results page.


5. From the Actions drop-down list, select View Property to display the Property Details page.


6. Click one of the following options:

·    CSV to save the list in a CSV format.

·    Excel to save the list in an Excel format.

·    XML to save the list in an XML format.

·    PDF to save the list in PDF format.
