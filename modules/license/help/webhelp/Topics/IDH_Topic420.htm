To Search the water works license perform the following steps:

1.      Click Hawker license on left navigator bar.

2.      Click Search to display the Hawker license search screen.

3.      Type the details as described in Table2.

 

 

                                        Table 2: Finding a Hawker License                              

Field

Description

Application Number

Enter the application number of the Hawker license.

Applicant Name

Enter the applicant name.

Application From Date

Type the date so that, the Hawker license created from this date will be displayed.

Application To Date

Type the date so that, the Hawker license created till this date will be displayed.

License Number

Enter the applicable License number.

Old license number

Enter the old license number.

Zone

Select appropriate Zone from the Zone drop down list.

Ward

Select appropriate ward from the Ward drop down list.

License Fee From

Enter License Fee from

License Fee To

Enter License Fee To

Hawker Category

From Hawker Category drop down list select Hawker category

Issued From Date

Enter Issued from date

Issued To Date

Enter Issued To date.

 

·         Search: To find the Hawker License depending upon Search Criteria.

·         Reset: To Reset all the values which entered.

·         Close: To close the existing screen.
