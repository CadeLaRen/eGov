Viewing Zone Wise Net Property Tax

Previous Top Next 

 

This report gives a high level report on the zone wise property usage and tax for the current financial year.

 

To view the zone wise net property tax sheet, perform the following steps:

1.   Click Property Tax from the left navigation bar.

2.   Click PTIS Reports.

3.   Click Zone Wise Prop Usage Demand. The tax details are displayed in a tabular form.

The following details are displayed:

·    Zone: The zone to which the property belongs. All the zones are listed. The tax  details are displayed against each zone.

·    Residential: The properties which are used for residential purposes.

·    Non Residential: The properties which are used for commercial purposes.

·    Mixed Usage: The properties which are used for both residential and commercial purposes. For example a multiple storey building of which some floors are used for offices and some are used for residence.

·    N/A Usage (Not Available Usage): The properties which do not have the usage details whether they are residential, commercial or mixed.

·    Aggregate Total: The total number of properties under each zone is displayed.

 

Under each type of a property the following details are displayed:

      

·    Number of Properties: The total number of properties under each type of  property in a particular zone.

·    Arrear Demand: An unpaid overdue tax prior to the current half yearly installment.

·    Current Demand: The tax to be paid  during the current half year.

·    Total Demand: The sum of the arrear tax and the current tax.

 

You can choose to print this report.

 

NOTE: To view ward wise net property tax, click the particular zone number. The tax details against each ward is displayed.
