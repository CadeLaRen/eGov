Create New Property tax

Previous Top Next 

To create a new Property tax perform the following steps:

1.      Log on to the Property as creator.

2.      Click Property tax on left navigation bar.

3.      Click New Property to display New Hospital license.

4.      Click Create New Property to display New License set up screen.

5.      Specify the License details as described in the following table:
