Modifying Property Address and Correspondence Details

Previous Top Next 

Occasionally, there may be clerical errors in the address that you have typed while registering a property. You can correct such errors at the request of the property owner after receiving necessary proof.


To modify an address, perform the following steps:

1.   Find the property  which you want to change the address of.

2.   From the Actions drop-down list, select Correct Property Particulars to display the Property Particulars page.

3.   From the Reason for Modification drop-down list, select one of the following:

·    Incorrect Address to modify the property address.

·    Change in Correspondence Address/Mobile or Email Address to modify the correspondence details.

You get the property address or the correspondence details that were entered while registering the property. You can make the necessary changes.

 

NOTE: You need to type a comment in the Comments field while modifying a property address.

 

4.   Click Submit to display the Property Confirmation page.                

5.   Click Confirm to save the changes  (or click Back to go to the previous page and make changes.).


A Property Acknowledgement page is displayed confirming the change of address.
