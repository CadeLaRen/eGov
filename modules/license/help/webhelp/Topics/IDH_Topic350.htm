Create Hawker License

Previous Top Next 

To create a new Hawker license perform the following steps:

1.      Log on to Hawker license as a creator.

2.      Click Hawker License on left navigation bar.

3.      Click Transactions to display New Hawker license.

4.      Click Create New license to display New License set up screen.

5.      Specify the License details as described in the following table:

 

 

                                                            Table 1: Module Set Up

Field

Description

Application Date

Type the Date on which the New Hawker license is created.

You can enter the date also by clicking the calendar icon.

Office Phone Number

Enter office phone number

Hawker Spot

Enter the Place of the hawker where it is situated

Hawker type

From the Hawker type drop list select appropriate hawker type

Zone

From the Zone drop down list Select appropriate zone.

Ward

From the ward drop down list select appropriate ward.

Applicant Details

 

Applicant Name

Enter the Applicant name.

Father or spouse name

Enter father or spouse name

Age

Enter Age of the applicant.

sex

Enter sex of the applicant.

Hawker Category

From the Hawker Category drop down list select appropriate hawker category

Remarks

Enter Comments if any.

Zone

From the Zone drop down list Select appropriate zone.

Ward

From the Ward drop down list Select appropriate ward

Plot No./House No.

Enter the plot number or House number.

Old House No.

Enter Old house number if any.

Remaining address

Enter complete addres if any thing missed in previous fields.

Pincode

Enter Pincode of the house address

Enter Residence of phone number

Enter Mobile Number

Enter Email id

Enter Uid

Upload a document from the system if any applicable.

Upload a Plumber photo who worked on it.

6.   Click Save to save the information.

7.   Click Close to exit from the screen.
