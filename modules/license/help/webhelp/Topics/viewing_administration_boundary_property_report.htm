Administration Boundary Reports

Previous Top Next 

You can use this report to find the details of a property across all zones and wards using various search criteria.


Examples:

·      You can find all independent buildings that come under the tax range of 5000 to 10000 rupees.

·      You can find all properties against which there is an arrear demand between 10000 to 15000 rupees.

·       You can find all residential properties whose area lie in between 10000 to 12000 square feet.


To view the report, perform the following steps:

1.   Click Property Tax from the left navigation bar.

2.   Click PTIS Reports.

3.   Click Administration Boundary Property Report  to display the data input page.

4.   Specify the following fields:

  

Field

Description

Zone

The zone to which the property belongs.

Ward

The ward to which the property belongs.

Street Name

The name of the street to which the property belongs.

Property Type

The type of the property such as:

·    Super Structure

·    Vacant Land

·    Independent Building

·    Flats in Multi Storied Building

·    Central Government 33%

·    Central Government 75%

·    State Government

Property Usage

The usage could be residential or non-residential.

Property Status

The current status of the property.

Sital Area

The total area of the plot.

Total Built Up Area

The total built-up area.

Current Demand

The half yearly tax charged on the property.

Arrear Demand (Rs.)

An unpaid overdue half yearly tax.

Total Demand (Rs.)

The sum of current demand and the arrear demand.


6.   Click  Search to display a list of properties with the following details:

·  Serial Number

·  Bill Number

·  Owner Name

·  Property Address

·  Usage

·  Current Demand

·  Arrears Demand

·  Total Demand

5.   Click one of the following options:

·  CSV to save the list in a CSV format.

·  Excel to save the list in an Excel format.

·  XML to save the list in an XML format.

·  PDF to save the list in PDF format.
