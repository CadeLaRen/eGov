Search File Notification

 

 

To Search the file perform the following steps:

1.      Log on to DMS as a creator.

2.      Click File Management on left navigation bar.

3.      Click Search file Notification to display the File Notification Search screen.

4.      Specify details as described in the following table:

 

 

                                                  Table 1: Module Set Up

Field

Description

File Details

File Number

A unique number that is associated with the file and it is auto generated

File Category

Select Category of the file from the File Category drop down list

File Heading

Enter File Heading to be searched

Department

Select respective department from Department drop down list

Zone

Select Zone from the Zone drop down list

Ward

Select Ward from the Ward drop down list.

File Received Date from

Enter File Received from date

File Received Date to

Enter File Received To date

File Status

Select Status of the file from File Status Drop down list

Received from/Sent to

Select Received from/sent to file from drop down list.t

Type

Select type of the file from type drop down list

 

 

File Sub Category

Select Sub Category of the file from the File Sub Category drop down list

Priority

Select the priority associated with the file. Priorities are:

·         Low

·         Medium

·         High

Outbound File Number

Enter Outbound File Number

 

 

·         Submit : To Submit the file.

·         Reset : To Reset all the values which entered.

·         Close : To close the existing screen.
