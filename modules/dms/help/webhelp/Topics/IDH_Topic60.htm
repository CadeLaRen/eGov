Create Inbound file

 

 

To create a new Inbound file System perform the following steps:

1.      Log on to DMS as a creator.

2.      Click File Management on left navigation bar.

3.      Click Create Inbound File to display New Inbound File.

4.      Specify the File and File movement details as described in the following table:

 

 

                                                  Table 1: Module Set Up

Field

Description

File Details

File Number

A unique number that is associated with the file and it is auto generated

File Date

Enter Date of Creation of the file

Department

Select the Concerned department from the Department drop down list.

Zone

Select Zone from the Zone drop down list.

Ward

Select Ward from the Ward drop down list.

Priority

Select the priority associated with the file. Priorities are:

·         Low

·         Medium

·         High

File type

Based on the File type hyperlink, this file type will be shown

Received/Sent Date

Enter File Received/Sent Date

File Category

Select the Category of the file which it belongs to from drop down list.

File Sub category

Select the sub category configured for a file category is shown in the drop down list.

File Heading

Enter Brief Description of the file

Summary of Document

Enter Detailed Description of the file.

Search Tags

Enter Tag associated with the file to facilitate searching of the file.

Attach Document

Attach document from the system if any applicable.

Comments

Enter Comments if any.

Sender Details

Sent From

Select the file Sent from details from Sent from drop down list.

Name of the sender

Enter the name of the person who has sent the file.

File Addressed to

Enter name of the person in the ULB to whom the file is addressed to

Address

Enter Address of the sender

Phone Number

Enter Phone Number of the sender

E-mail

Enter mail id of the sender

Forward To

Receiver Department

Select Receiver Department from Receiver Department Drop down list

Receiver Designation

Select Receiver Designation from Receiver Designation drop down list

Receiver

Select Receiver name from Receiver drop down list

 

·         Save as Draft : To save the file as Draft

·         Submit : To Submit the file.

·         Reset : To Reset all the values which entered.

·         Close : To close the existing screen.
