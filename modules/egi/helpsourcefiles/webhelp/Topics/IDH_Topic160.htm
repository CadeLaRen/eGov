Creating a User and Assigning a Role to the User

 

To create a user, perform the following steps:

1.           Log on to the Administration System.

2.           Click User Management from the left navigation bar.

3.           Click Create User to display the Create New User page.

4.           Specify the details as described in the following table:

 

Table1: User Details

Field

Description

First Name

Enter the first name of the user.

Middle Name

Enter the middle name of the user.

Last Name

Enter the last name of the user.

From date

Enter the date (DD/MM/YYYY) from which the user has access to the application. You can also click the Calender button  and specify a date.

To date

Enter the date (DD/MM/YYYY) till which the user has access to the application. You can also click the Calender button  and specify a date.

Salutation

Enter the salutation as either Mr or Mrs or Ms.

Date of Birth

Enter the date (DD/MM/YYYY) of birth of the user. You can also click the Calender button  and specify a date.

User Name

Enter a login ID for the user.

Password

Enter a password for the user.

Repeat Password

Re-enter the password.

Is Active

Select this check box to activate the user. If not activated, the user cannot log on to the application.

 

6.   To assign a role to the user, specify the following details in the user roles section:

 

Table2: User Roles

Field

Description

Select User Role

Select a role from the drop-down list.

From date (dd/mm/yyyy)

Enter the date from which the user has been assigned the role.

To date(dd/mm/yyyy)

Enter the date till which the user has been assigned the role.

 

 NOTE: If there are multiple user roles, click the Add Row button and specify the details. Similarly, click the Delete Row button to delete a user role.

 

7.   Click Submit.

You get a confirmation that the user has been created successfully.
