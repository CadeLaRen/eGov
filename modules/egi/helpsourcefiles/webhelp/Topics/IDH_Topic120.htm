Changing Password

 

To change the password, perform the following steps:

1.   Log on to the Administration System.

2.   Click the Change Password hyperlink located at the top right side of the page. You get the Change Password page with the following fields:

·    Username -this cannot be edited

·    Old Password

·    New Password

·    Confirm New Password

3.   Enter the old password in the Old Password field.

4.   Enter the new password in the New Password field.

5.   Enter the new password again in the Confirm New Password field.

6.   Click Submit. The password is then changed.
