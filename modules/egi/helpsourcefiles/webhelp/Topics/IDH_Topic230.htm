Viewing User Counter Map Reports

 

To view a Location or a Terminal, perform the following steps:

1.   Log on to the Administration System.

2.   Click Reports from the left navigation bar.

3.   Click UserCounter Reports to display the User Counter Map Reports  page as shown below:

 

Figure 1 : User Counter Map Reports page

 

4.   Select one of the following options:

·    Location: to view Location based reports

·    Terminal: to view Terminal based reports

5.   Specify the following details:

·    From Date (dd/mm/yyyy): Enter the date from which onwards you want to view the reports.

·    To Date (dd/mm/yyyy): Enter the date till which you want to view the reports.

6.   Click Submit to display the reports.
