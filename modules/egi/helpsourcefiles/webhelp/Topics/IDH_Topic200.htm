Creating a Location or a Terminal

 

Location refers to an office that has various counters. Each counter is represented by a computer. All computers are interconnected and as well are connected to a router which communicates with the ULB server over Internet. Therefore, all computers in a location are identified by the IP Address of the router. There is no unique IP Address to identify a specific computer in a location.

 

Example:

·         Location : Indiranagar

·         Counters: Counter 1 (computer 1-IP Address: 192.168.1.1), Counter 2(computer 2-IP Address: 192.168.1.1), .....Counter n (computer n-IP Address: 192.168.1.1).

 

Terminals are computers which are directly connected to the ULB server. Each terminal is represented by a computer, which is identified by a unique IP Address.

 

Example:

·         Terminals: Terminal 1 (computer 1-IP Address: 192.168.1.1), Terminal 2 (computer 2-IP Address: 192.168.1.2), .....Terminal n (computer n-IP Address: 192.168.1.3).

 

A User of a specific location will not be able to log on to the system from a different location, since the IP Address of each location is unique. Similarly, a user of a specific terminal will not be able to log on to a different terminal. A user of a counter can log on to any other counters of the same location, since all counters of a specific location will have the same IP Address.

 

To create a Location or a Terminal, perform the following steps:

1.   Log on to the Administration System.

2.   Click Location from the left navigation bar.

3.   Click Create Location/terminal to display the Location page as follows:

 

Figure 1 : Location page

 

4.   Click Create  to display the Create Location/terminal page as follows:

 

Figure 2 : Create Location/Terminal page

 

5.   Select any one of the following options:

·    Location: To create a location. Specify the details as described in Table 1

·    Terminal: To create a terminal. Specify the details as described in Table 2

Table1: Create Location

Field

Description

Location Name

Enter the name of the location.

Active

Select the check box to activate the location.

Description

Enter a brief description about the location

IP Address

Enter the IP address allotted to the location.

There can be multiple IP addresses.

 

NOTE: Click Add Row to enter another IP address. Click Delete Row to delete a row. The first row cannot be deleted.

 

Counter

Enter the name of the computer.

 

NOTE: Click Add Row to enter the details of another counter. Click Delete Row to delete a row. The first row cannot be deleted.

 

Description

Enter a brief description about the counter.

 

Table2: Create Terminal

Field

Description

Location Name

Enter the name of the place where the terminal is located.

Active

Select the check box to activate the terminal.

Description

Enter a brief description about the location

Counter

Enter the name of the computer.

 

NOTE: Click Add Row to enter the details of another counter. Click Delete Row to delete a row. The first row cannot be deleted.

 

Description

Enter a brief description about the counter.

IP Address

Enter the IP address of the terminal.

 

6.   Click Save.

You get a confirmation message that a terminal or a location has been created.
