Setting Master Data Values of a Module

 

To set the master data values of a module, perform the following steps:

1.   Log on to the Administration System.

2.   Click Configuration from the left navigation bar.

3.   Click Update to display the Create page as shown below:

 

Figure 1 : Update page

 

4.   Select a module from the Select Module Name drop down list.

5.   Click Update to display the Master Set Up page. The description about the value to be entered is shown.

6.   Enter the following details:

·    Value: enter the master value.

·    Effective From: the date from which onwards the value entered is valid.

For example, if the module selected is New Property and the description is Building Land, then the Value can be Temple Land, Central Government Land, State Government Land, Church Land, Private Land, Chennai Corporation, Others.

 

 NOTE: Click the Plus button  to add a row to enter more values.

 Click the Minus button  to delete a row.

 

7.   Click Save.

You get an acknowledgment that the key value has been created successfully.
