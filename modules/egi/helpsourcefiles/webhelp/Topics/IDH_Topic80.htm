Logging On to the Administration System

 

To log on to the Administration System, perform the following steps:

1.   Log on to the eGov portal. The login page is displayed as follows:

 

Figure 1 : Login page

 

2.   Enter your user name in the Name/User Name field.

3.   Enter your password in the Password field.

4.   Click Login to display the eGov home page.

5.   Click Administration from the left navigation bar to display the Administration System home page.
