Creating a Root Boundary

 

A Root Boundary is the parent boundary, which means that it can have child boundaries under it. For example, if Hierarchy Type is Administration, and Boundary Type is City, then Chennai can be the Root Boundary.

 

To create a Root Boundary, perform the following steps:

1.   Log on to the Administration System.

2.   Click Jurisdiction from the left navigation bar.

3.   Click Create Boundary to display the Search Boundary page.

4.   Select a Hierarchy Type from the Hierarchy Type drop-down list.

5.   Select a Boundary Type under which you want to create a Root Boundary from the Boundary Type drop-down list.

6.   Click Search.

7.   Click Create Root to display the Create Boundary page.

8.   Enter the details as described in the following table:

 

Table 1: Create Boundary

Field

Description

Hierarchy Type

This field is prepopulated with prior selection from the Hierarchy Type drop-down list .

Boundary Type

this field is prepopulated with prior selection from the Boundary Type drop-down list.

Boundary Name

Enter the name of the Root Boundary that you want to create.

Boundary Name Local

Enter the name of the Root Boundary that you want to create in local language.

Boundary Number

Enter the number allotted to the boundary. This number is unique within the parent boundary selected.

From date

Enter the date (DD/MM/YYYY) from which onwards the Root Boundary should exist.

To date

Enter the date (DD/MM/YYYY) till which the Root Boundary should exist.

City Logo

Enter the name of the image file of the logo to be displayed.

Is Active

Select to activate the Root Boundary.

City Base URL

Enter the city Web site link.

 

9.   Click Save.

You get an acknowledgement that the boundary has been successfully created.

 

NOTE: You cannot create a Root Boundary for a Child Boundary Type.
