Finding a User

 

To find a user, perform the following steps:

1.   Log on to the Administration System.

2.   Click User Management from the left navigation bar.

3.   Click Search User to display the Search User page.

4.   Enter the user name in the User Name field.

5.   Click Submit.
The details of the user are displayed in a grid.

 

You can do the following:

·      Assign a boundary to the user

·      View a user's jurisdiction

 

For more information, refer to the following topics:

·      Assigning a region to the User or Modifying a User Jurisdiction

·      Viewing a User Jurisdiction
