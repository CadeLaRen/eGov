Glossary

 

Base URL: The Web site is made up of a series of web pages. Each web page has a full Uniform Resource Locator (URL), for example, http://www.Administration.com/intro.htm. The Webpage URLs usually start with the same set of letters, for example, http://www.Administration.com/ in the above example. This common prefix is called as the Base URL.

 

Internet Protocol (IP) Address: It is a unique numeric label used to locate and identify a device over a network. For example, 192.168.255.255 is an IP address.          

 

Network Topology: It the physical interconnections of the elements such as links, nodes and peripherals of a computer network.

 

Parameter Query : A parameter query is a query that prompts you for a value, or values, each time you run it. The value that the query asks for can be text, a date, a number, or just yes or no. A query that requires you to provide one or more criteria values. For example, you could enter the expression Between [Start date:] And [End date:] in the criteria cell for a date field. Start date: and End date: will be displayed when you run the query. The dates that you enter will then be used as the parameters in the Between/And expression.

 

Router: It is an electronic device used to connect two or more computers or other electronic devices to each other, and to the Internet, by wire or radio signals. This allows several computers to communicate with each other and to the Internet simultaneously.
