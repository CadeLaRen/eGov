Viewing, Modifying or Deleting a Hierarchy Type

 

To view, modify or delete a Hierarchy Type, perform the following steps:

1.   Log on to the Administration System.

2.   Click Jurisdiction from the left navigation bar.

3.   Click  Create Hierarchy Type to display the Hierarchy Types page.

4.   Select a Hierarchy Type that you want to view or modify or delete from the Hierarchy Type drop-down list.

5.  You can do one of the following:

a) Click View to display the View Hierarchy page.

NOTE: Click Back to go to the Hierarchy Types page.
 

b) Click Modify to display the Update Hierarchy page. Edit the data in the associated fields and then click Save to update the information.

NOTE: You get a confirmation that the Hierarchy Type has been updated successfully. Click Back to go to the Hierarchy Types page.
 

c)  Click Delete to delete a Hierarchy Type.

NOTE: You get a confirmation that the Hierarchy Type has been deleted successfully. Click Back to go to the Hierarchy Types page.

You cannot delete a Hierarchy Type if there is a Boundary type that belongs to the Hierarchy Type.
