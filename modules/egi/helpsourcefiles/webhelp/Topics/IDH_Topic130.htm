Creating a Role

 

To create a role, perform the following steps:

1.   Log on to the Administration System.

2.   Click User Management from the left navigation bar.

3.   Click Create Role to display the Roles page.

4.   Click Create to display the Create Role page.

5.   Enter the details as described in the following table:

Table1: Create Role

Field

Description

Role Name

The name of the role or a designation. For example, AE.

Role Desc 

A brief explanation of the role. For example, Assistant Engineer.

Role Name Local

The name of the role or a designation. For example, AE in local language

Role Desc Local

A brief explanation of the role. For example, Assistant Engineer.

 

7.   Click Save.


You get a confirmation that the role has been created successfully.
