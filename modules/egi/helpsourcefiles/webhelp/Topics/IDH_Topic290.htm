Adding a Child Boundary Type

 

To add a Child Boundary Type, perform the following steps:

1.   Log on to the Administration System.

2.   Click Jurisdiction from the left navigation bar.

3.   Click Search Boundary Type to display the View Boundary Type page.

4.   Select a Hierarchy Type from the Hierarchy Type drop-down list.

5.   Select a Boundary Type under which you want to add a Child Boundary Type from the Boundary Type drop-down list.

6.   Click Add Child Boundary to display the Boundary Type page.

7.   In the Boundary Type field, enter the name of the Child Boundary.

 

NOTE: The Hierarchy Type and Parent Boundary Type fields are populated with your prior selection from the Hierarchy Type and Boundary Type drop-down lists.

 

8.   In the Boundary Type Local field, enter the name of the Child Boundary in local language.

9.   Click Add Child.

You get an acknowledgement that the Boundary Type has been created successfully.
