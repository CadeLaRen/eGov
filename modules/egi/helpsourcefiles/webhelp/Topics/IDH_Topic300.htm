Adding a Child Boundary to the Root Boundary

 

A Child Boundary is a subset of the Root Boundary. For example, if Hierarchy Type is Administration, Boundary Type is City, and the Root Boundary is Chennai, then the Child Boundary can be Korrukupet.

 

To add a Child Boundary to the Root Boundary, perform the following steps:

1.   Log on to the Administration System.

2.   Click Jurisdiction from the left navigation bar.

3.   Click  Create Boundary to display the Search Boundary page.

4.   Select a Hierarchy Type from the Hierarchy Type drop-down list.

5.   Select a Boundary Type from the Boundary Type drop-down list.

6.   Select a Root Boundary under which you want to add a Child Boundary from the Boundary Values drop-down list.

7.   Click View to display the boundary details.

8.   Click Add Child Boundary.

9.   Enter the details as described in the following table:

 

Table 1: Create Boundary

Field

Description

Boundary Name

Enter the name of the Child Boundary that you want to create.

Boundary Number

Enter the number allotted to the Child boundary. This number is unique within the parent boundary selected.

From date

Enter the date (DD/MM/YYYY) from which onwards the Root Boundary should exist.

To date

Enter the date (DD/MM/YYYY) till which the Root Boundary should exist.

 

10.       Click Save.

You get an acknowledgement that the boundary is created successfully.
