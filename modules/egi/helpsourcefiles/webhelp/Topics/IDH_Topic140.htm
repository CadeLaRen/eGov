Viewing, Modifying, or Deleting a Role

 

To view, modify, or delete a user role, perform the following steps:

1.   Log on to the Administration System.

2.   Click User Management from the left navigation bar.

3.   Click Create Role to display the Roles page.

4.   Select a user role that you want to  view or modify or delete from the Role Name  drop-down list.

5.  You can do one of the following:

a) Click View to display the View Role page with the user role details.

NOTE: Click Back to go to the Roles page.
 

b) Click Modify to display the UpdateRole page. Edit the data in the associated fields and then click Save to update the information.

NOTE: You get a confirmation message that the role has been updated successfully. Click Back to go to the Roles page.
 

c)  Click Delete to delete a role.

NOTE: You get a confirmation message that the role has been deleted successfully. Click Back to go to the Roles page.
