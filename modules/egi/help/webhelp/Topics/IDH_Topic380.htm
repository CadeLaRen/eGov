Introduction

 

eGov ERP Suite is Web-based and can be accessed from anywhere. In order to reach out the people to deliver better service, ULBs set up locations in various parts of the city, in addition to the counters available in the headquarters.

 

For example, ULB can set up multiple collection centers in various parts of your city to enable citizens to pay property tax on time. In each center, there may be multiple counters.

 

As a system administrator, you need to set up various locations and counters. After doing this, you need to map a user to a counter.
