Introduction

 

Before using any of the modules of the eGov ERP Suite, System administrators need to set up the system catering to the requirements of your ULB. The setup is performed using the eGov Administration module, which is again a part of the ERP Suite.

 

For example, assume that you have deployed eGov ERP Suite in your ULB. Now, you need to identify the employees who are going to use the system and create a login account for each of them. You may need to create different departments for the ULB. Not all employees may have the privilege to use all features in a module, in which case, you need to associate a feature with a role.

 

The eGov Administration module acts as a backbone of the eGov ERP Suite.

 

Key Features

 

Using eGov Administration module, you can:

·         Create new users

·         Add a new modules or features

·         Create a new department

·         Create a position hierarchy

·         Set up roles

·         Set up modules and features

·         Create designations and positions

·         Enforce role-based access control

·         Set up administrative boundaries
