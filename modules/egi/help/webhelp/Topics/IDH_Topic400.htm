Modifying User Details and User Role

 

Occasionally, you may need to modify the details associated with a user such as the name, user ID, password, and role name.

 

To modify a user's details, perform the following steps:

1.   Find the user whose details you want to modify.

2.   Click the user name to display the Update New User page with the user's details.

3.   Edit the user's details as described in Table1: User Details.

4.   Select the Modify check box to edit the user role details.

5.   Edit the user role details as described in Table2: User Roles.

 

 NOTE: If there are multiple user roles, click the Add Row button and specify the details. Similarly, click the Delete Row button to delete a user role.

 

6.   Click Submit.

You get a confirmation that the user details have been modified successfully.
