Modifying an Action

 

To modify an action, perform the following steps:

1.   Log on to the Administration System.

2.   Click Module from the left navigation bar.

3.   Click Module from the left navigation bar.

4.   Click View/Update  to display the Update Module OR Create/Update Action page.

5.   Click the Browse button  to select the module that you want to modify. You get a Search screen Module page.

6.   Click the name of the module that you want to modify. The Module Name field gets populated with the selected module.

NOTE: Click the Plus button  to select the submodules.
 

7.   Click Create/Update Action to display the Action Set up page.

8.   Modify the action details as described in Table1: Action Set Up.

9.   Click Save.
You get a confirmation message that the action has been updated successfully.
