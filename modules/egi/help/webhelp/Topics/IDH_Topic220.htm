Assigning a User to a Location or a Terminal

 

To assign a user to a location or a terminal, perform the following steps:

 

1.   Log on to the Administration System.

2.   Click Location from the left navigation bar.

3.   Click User Counter Mapping to display the Map User to Counter page.

4.   Select any one of the following options:

·    Location: To assign a user to a location. For example, a location can be an office which has many computers or terminals.

·    Terminal: To assign a user to a terminal. A terminal is a device providing access to a computer or is a computer itself.

5.   Select a location or a terminal from the Location/Terminal drop-down list.

6.   Select the Show All option. This shows the history of assignments of the user you select.

7.   Click Submit. A grid is displayed with the assignment details if a user has been assigned.

8.   Specify the following details:

·    User: Enter the user name of the person whom you are assigning a counter or a terminal.

·    From Date (dd/mm/yyyy): Enter the date from which the user be assigned a location or a terminal.

·    To Date (dd/mm/yyyy): Enter the date till which the user be assigned a location or a terminal.


NOTE: Click Add Row to enter the details of another user.
 

9.   Click Save.
