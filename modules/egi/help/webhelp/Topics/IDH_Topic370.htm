Introduction

 

After deploying the eGov ERP Suite in your ULB, you must identify the users of each ERP module and create login accounts for each of them. User names may vary from one module to another. A user name of the eGov EIS module can be "Employee", while a user name of the eGov PTIS module can be Assessor or Tax Collector.

 

Each user of the eGov ERP Suite can be associated with a designation (role).  Each designation can be associated with a set of features/actions. For example, a person with the Employee role can access the EIS module to create a leave application, while the person with the Manager role can access the EIS module to approve the leave.

 

Also, the jurisdictional area of a user differs from role to role. For example, a tax collector may have the privilege to collect property tax only for a specific ward, while the ARO may have the privilege to collect property tax for a specific zone, and the commissioner may have the privilege to collect tax across the city.

 

When you assign a jurisdictional area to a user, functional area of the user will be limited to the defined jurisdictional area.
