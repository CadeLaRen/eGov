Assigning a Jurisdiction to a User

 

To assign a jurisdiction to the user, perform the following steps:

1.   Find the user to whom you want to assign a jurisdiction.
The details of the user are displayed.

2.   Click  Edit Jurisdiction to display the user jurisdiction page.

3.   Specify the details as described in the following table:


Table1: User Jurisdiction

Field

Description

Boundary Type Value

Do the following steps:

a)          Click the Browse button  to select a boundary under which the user should have jurisdiction. You get a Search screen Boundary page.

b)          Click the name of the boundary.The Boundary Type Value field gets populated with the selected boundary.

NOTE: Click the Plus button  to select the sub boundaries.
 

From date(dd/mm/yyyy)

Enter the date from when the user will have the jurisdictional power

To date(dd/mm/yyyy)

Enter the date till when the user will have the jurisdictional power

NOTE: Click Add Row to assign another boundary to the user.
 

 

4.   Click Save.

You get an acknowledgment that the user jurisdiction has been updated successfully.
