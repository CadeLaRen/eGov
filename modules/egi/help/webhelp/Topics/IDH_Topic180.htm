Viewing the Jurisdictional Area of a User

 

A user's functional area may be limited to a specific area.

 

To view a user jurisdiction, perform the following steps:

1.   Search the user whose jurisdiction areas you want to view. For more information, see Searching a User.

2.   Click  View Jurisdiction to display the User Jurisdiction page.
The following details are displayed in a grid:
 

·  User Name: the user name of the user

·  Boundary Type Value: the name or the number of the zone/ward to which the user has jurisdiction

·  From Date: the date from when the user has jurisdiction over the boundary.

·  To Date: the date till when the user has jurisdiction over the boundary.
