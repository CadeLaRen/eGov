Setting Up an Action for a Module

 

To set up an action for a module, perform the following steps:

1.   Log on to the Administration System.

2.   Click Module from the left navigation bar.

3.   Click Module from the left navigation bar.

4.   Click View/Update  to display the Update Module OR Create/Update Action page as follows:

Figure 1: Update Module OR Create/Update Action page

 

5.   Click  the Browse button  to select the module for which the action has to be set up. You get a Search screen Module page.

6.   Click the name of the module for which the action has to be set up. The Module Name field gets populated with the selected module.

NOTE: Click the Plus button  to select the submodules.
 

7.   Click Create/Update Action to display the Action Set up page.

8.   Specify the action details as described in the following table:

Table 1: Action Set Up

Field

Description

Action Name

Enter the name of the action.

Base URL

Enter the Base URL of the Action. For more information, see Glossary.

Module Description

Enter a brief description about the module.

Query Parameters

Enter the value to be passed to the end of the Base URL. For more information, see Glossary.

URL Order Id

TBD

Order Number

Enter the number according to which the Action should appear on the screen.

Display Name

Enter the name of the action to be displayed. For example, Create Property.

Help URL

Enter the link of the Online Help.

Is Enabled

Select or clear the checkbox to activate or deactivate the action.

Delete

Click the Delete button to delete an Action

NOTE:  Click Add to set up another action.

9.   Click Save to save the information.

You get a confirmation message that the action has been created successfully.
