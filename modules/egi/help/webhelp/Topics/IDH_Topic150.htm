Assigning an Action to a Role

 

There are various types of users who need to access the Application for different purposes. For instance, at the ULB level, the accountant uses it for day-to-day accounting of vouchers; while the Accounts Superintendent uses to check a bank balance; and the Commissioner may use it to view reports such as the Income & Expenditure account or budget variance report to assess the performance of the ULB.

 

To assign an action to a role, perform the following steps:

1.   Log on to the Administration System.

2.   Click User Management from the left navigation bar.

3.   Click Role Action Mapping to display the Role Action Setup page as follows:
                            

Figure 1: Role Action Setup page

 

5.   Click the Browse button  to select the module for which the action has to be set. You get a Search screen Module page.

6.   Click the name of the module for which the action has to be set. The Module Name field gets populated with the selected module.

NOTE: Click the Plus button  to select the submodules.
 

7.   Select a role to which the action has to be set from the Role Name drop-down list.

8.   Click Setup Action to display the Role Action Setup page.

9.   Select or clear to activate or deactivate the actions displayed.

 

NOTE: The actions displayed are based on the module and the role selected.

 

10.       Click Save to save the information.
