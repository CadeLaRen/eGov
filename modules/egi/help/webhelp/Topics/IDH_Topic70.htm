Creating a Department

 

To create a department, perform the following steps:

1.   Log on to the Administration System.

2.   Click Department from the left navigation bar.

3.   Click Create Department to display the Department page.

4.   Click Create to display the Create Department page.

5.   In the Department Name field, enter the name of the department that you want to create. Example: Engineering Department.

6.   In the Department Code field, enter a code to uniquely identify the department.

7.   In the Department Details field, enter additional details of the department.

8.   In the Department Address field, enter the address of the department.

 

NOTE: Select the Is Billing Location check box if the department to be created is a billing counter.

 

10.       Click Save to save the information.
You get a confirmation that the department has been created successfully.
