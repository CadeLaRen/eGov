Viewing Master Data Values of a Module

 

To view the master data values of a module, perform the following steps:

1.  Log on to the Administration System.

2.  Click Configuration from the left navigation bar.

3.  Click View to display the View page as follows:

 

Figure 1 : View page

4.  Select a module that you want to view from the Select Module Name drop-down list.

5.  Click View to display the Master SetUp page with the master data details.
